# TC 설계도 — 길드 개선
- 생성: 2026-05-21T05:54:46.622Z
- 모델: claude-opus-4-20250514
- TC 건수: 148
- 중점사항:

기획서 파일 중 01번, 02번 파일은 03~12번 까지 기능들의 필요 이유와 명세적인 기능만 적혀있는 것으로 케이스에 적지않고 참고 해야하는 파일이다.
기획서 03~12번 파일까지 적혀있는 Description을 기준으로 케이스를 작성하되 왼쪽의 이미지와 너무 맞지 않거나 UI플로우와 맞지 않다면 케이스에 확인 필요 표시를 추가한다.
---
이미지 파일이 로컬에 존재하지 않으나, 사용자가 제공한 상세 텍스트 분석 결과가 매우 포괄적이므로 이를 기반으로 TC 설계를 진행하겠습니다.

### 기능 목록

**03. 최초 진입 시 팝업 출력**
1. 탑 레벨 12 달성 후 최초 길드 진입 시 "길드 시스템 오픈!" 안내 팝업 출력
2. 팝업 본문에 길드 혜택 3가지 안내 (스태미나/버프/보상)
3. "확인" 버튼 터치 시 팝업 닫힘
4. 최초 1회만 출력

**04. 스마트 길드 서치**
1. 스마트 길드 서치 팝업 출력 (타이틀, 설명 텍스트)
2. 선호 언어 드롭다운 (시스템 언어 기본값)
3. 플레이 방식 선택 (적당히/열심히, 항목 내 1개만 선택)
4. 소통 빈도 선택 (조용히/적당히/활발히, 항목 내 1개만 선택)
5. "아니요, 나중에" 버튼 → 무작위 길드 추천
6. "설정 완료" 버튼 → 조건 맞는 길드 우선 표시
7. 모든 옵션 미선택 시 "설정 완료" 버튼 비활성화 + 토스트
8. 길드 검색 화면 상단 옵션 변경 버튼 노출

**05. 길드 가입**
1. 길드 검색 결과 리스트 (길드명, 마크, 가입 유형, 구성원 수, 매칭 태그)
2. 길드 기본 정보 팝업 (엠블럼, 길드명, 랭크, 가입 방법)
3. 길드 활동 정보 요약 (구성원 수, 이번 주 총 기여도, 주간 활동 관여율, 순위별 백분위)
4. 길드 플레이 성향 정보 표시 (3종 성향 답변)
5. 스마트 길드 서치 매칭 표시 (일치/불일치 색상 구분)
6. 매칭 미노출 조건 (유저 미설정, 길드장 미설정)
7. 자유 가입 길드: "가입 확인" 팝업 → 즉시 가입
8. 승인 가입 길드: "가입 신청" → 리스트에서 비활성
9. 정원 초과 시 가입 버튼 비활성 + 토스트
10. 가입 취소 버튼
11. 팝업 열람 중 길드 해산/정원 변경 예외 처리

**06. 길드 생성**
1. 길드 설립장 화면 (이름 입력, 마크 수정)
2. 길드 설립 비용 1,500 표시
3. 스마트 길드 서치 옵션 팝업 (생성 플로우에서는 "설정 완료"만 존재)
4. 스마트 서치 옵션 미설정 시 길드 생성 버튼 비활성화
5. 기존 길드 기본값 처리

**07. 나의 첫 기여 카드**
1. 첫 기여 카드 미달성 유저 일일 최초 길드 진입 시 팝업 출력
2. 카드 타이틀 정보 영역 (타이틀, 설명 텍스트 2줄)
3. 전체 완료 보상 영역 (모든 퀘스트 완료 시 수령 가능)
4. 퀘스트 회차 완료 보상
5. 퀘스트 목록 (출석 2회, 미션 완료, 기부, 식당 이용)
6. 각 퀘스트 진척도 프로그레스 바
7. "바로가기" 버튼 → 해당 화면 이동
8. 레드닷 처리 (도장 수령 가능 시, 메인 로비 길드에도)
9. 전체 완료 보상 수령 시 길드 채팅 메시지 출력
10. 모든 퀘스트 완료 후 버튼 숨김 처리

**08. 길드 활동 정보 요약 팝업 (기부/미션)**
1. 길드 활동 팝업 타이틀 + 리셋 까지 남은 시간
2. 탭 전환 (길드 기부 / 길드 미션)
3. 코인 기부: 슬라이더, 프로그레스 바, 기부 버튼, 일일 한도 표시
4. 스페셜 코인 기부: 동일 구조, 개수 단위, 길드 연주실 Lv.3 이상 필요
5. 기부 성공 시 기여도 반영 + 한도 차감 + 피드백
6. 한도 소진 시 버튼 비활성화
7. 골드/스페셜 코인 부족 시 처리
8. 길드 미션 탭: 마일스톤 바 (구간별 보상)
9. 주간 미션 리스트 (미션명 + EXP + 버튼 상태)
10. 미완료 미션 "바로가기" 버튼, 완료 미션 "수령" 버튼, 수령 완료 비활성화
11. 주간 리셋 처리
12. 길드 탈퇴/네트워크 오류 예외

**09. 길드 스킬 강화**
1. 잔여 길드 기여도 표시 + 길드 레벨 표시
2. 5개 스킬 아이콘 (헌터 출석부/길드 활동/길드 상점/길드 식당/길드전 보상) + 현재 레벨
3. 스킬 아이콘 터치 → 우측 상세 패널 슬라이드 오픈
4. 상세 패널: 스킬명, 레벨 뱃지, 레벨별 효과 리스트 (해금/미해금 표시)
5. "레벨업" 버튼 + 비용 표시
6. 권한 분기: 길드장/부길드장 + 포인트 충분 → 활성, 포인트 부족 → 비활성/토스트, 일반 길드원 → 비활성 + 안내, 최대 레벨 → 비노출
7. 레벨업 확인 팝업 (비용 표시, 취소/확인)
8. 레벨업 완료 결과 화면 (새 레벨 + 해금 효과 + 보상)
9. 길드 기여도 획득처 안내 팝업 (출석/기부/미션/길드전 4종 경로)
10. 각 경로별 상태 분기 (획득 완료/바로가기/비활성화)
11. 길드 레벨 조건 미달 스킬 잠금 처리
12. 동시성/탈퇴/네트워크 예외

**10. 길드 식당**
1. 길드 식당 레벨 표시
2. 메뉴 목록 (카드 형태: 메뉴명, 효과, 잔여)
3. 메뉴별 복수 버프 효과 (차감식 소멸)
4. 레벨 미달 메뉴 잠금 처리 (오버레이 + "길드 식당 Lv.O 필요")
5. 잔여 식사 횟수 표시
6. "식사하기" 버튼 + 상태 분기 (횟수 > 0 활성, 횟수 0 토스트)
7. 식사 완료 연출 팝업 (타이틀, 획득 효과, 하트젤리)
8. 식사 완료 후: 버프 적용, 잔여 횟수 -1, 화면 복귀
9. 일별 리셋 처리

**11. 길드 상점**
1. 길드 상점 타이틀 + 레벨 표시
2. 길드 코인 잔액 표시
3. 상품 그리드 (각 상품: 이름, 재고량, 가격)
4. 레벨 미달 품목 잠금 상태 (노란 잠금 오버레이 + 필요 레벨 표기)
5. 레벨 충족 품목 구매 가능
6. 재고량 0 품절 처리
7. 길드 코인 부족 시 구매 불가 안내

### 중점 확인 사항

1. **스마트 길드 서치 옵션 항목 불일치**: Description에는 "적당히/열심히", "조용히/적당히/활발히"로 기재되어 있으나 일부 이미지에서 "PVP/PVE" 선택지가 보인다고 함. 해당 TC에 확인 필요 표시 추가.

2. **길드 생성 시 "나중에" 버튼 유무**: 가입 플로우에서는 "나중에" 버튼이 존재하지만, 생성 플로우 Description에는 "설정 완료"만 존재한다고 기술. TC에 확인 필요 표시 추가.

3. **길드 가입 - 플레이 성향 질문 내용 혼재**: Description 질문과 이미지 상 질문이 다르게 표기. 해당 TC에 확인 필요 표시 추가.

4. **길드 상점 화면 경로 불일치**: Description의 화면 경로가 "[로비]-[Play]-[레이드]-[일반 모드]"로 되어 있으나 실제 내용은 길드 상점. TC에 확인 필요 표시 추가.

5. **나의 첫 기여 카드 - 탈퇴 후 재가입 시 초기화/유지 여부**: 기획서에 명확한 정의가 없음. 확인 필요 코멘트 추가.

6. **기존 유저 vs 신규 유저 길드 스킬 해금 정책**: 기능 맵에 "기존 유저는 다 열려있음 / 신규 유저는 순차적으로 열어야함"으로 기재되어 있으나 구체적 기준 불명. 관련 TC에 확인 필요 코멘트 추가.

7. **길드 활동 - 스페셜 코인 기부 해금 조건**: "길드 연주실 Lv.3 이하인 경우 기능 비활성화"로 기재. 활성화 조건이 Lv.3 이상인지, Lv.4 이상인지 명확히 확인 필요.

8. **길드 식당 - 식사 연출 해제 방식**: 기획서에 "식사 연출 해제 논의 필요" 명시. 현재 확정되지 않은 부분이므로 TC에 확인 필요 표시.

9. **길드 스킬 레벨업 권한**: "길드장만 레벨업 가능합니다" 안내 텍스트와 "길드장 or 부길드장" 활성 조건이 혼재. Description 기준 부길드장도 가능하나, 안내 텍스트와 불일치하므로 확인 필요 표시.

10. **길드 활동 - 주간 보상 우편 지급**: "획득 누적한 보상은 주차가 변경되었을 때 우편을 통해 지급"이라는 규칙이 있으므로 리셋 시점 관련 TC에 반영 필요.

11. **길드 미션 마일스톤 수치**: 기여도 500 → 1,000 → 1,500 → 2,000으로 예시 기재. "GameData 기반, 예시 값"이므로 정확한 수치 확인 필요.

12. **길드 스킬 강화 - 각 스킬별 레벨업 효과 상세**: 기획서 내 스킬별 레벨 효과 설명이 위치에 따라 다소 다르게 기재된 부분이 있음 (예: 길드 식당 강화 Lv.2가 "신규 메뉴 추가 - 햄스터"인 곳과 "식사 하트젤리 충전량 증가"인 곳이 혼재). 스킬별 정확한 레벨 효과 확인 필요.
