## 기능 목록

### 1. 레이드 모드 진입 화면
- 싱글/멀티 전환 토글 버튼 (Default: 싱글)
- 등장 보스 정보 표시 (보스 포트레이트, 명칭, 드랍 정보, 배경)
- 6종 보스별 차별화된 슈퍼에픽 장비 드랍 표시
- 카테고리 리스트 (불타는 곳 레이드, 성장 던전, 탑의 균열, 어비스 던전)
- 잔여 입장 횟수 표시
- 입장하기 버튼

### 2. 레이드 모드 로비
- **싱글 모드**: 1인 캐릭터 + 보스 등장
- **멀티 모드**: 2인 매칭 (기존 4인에서 축소), 지인 초대 방식
- 보스/난이도 변경 버튼 및 팝업
- 빠른 난이도 변경 버튼 (1-8단계)
- 입장 전투력, 추천 속성, 입장 조건 표시
- 단계별 잠금/해제 시스템
- 보스 변경 기능 (stage_id 순서)
- PLAY 버튼

### 3. 레이드 인게임  
- 인게임 드랍 연출 변경 (약 3초)
- **제거된 기능들**: 코코방울 연출, 코코방울 축복, 트로피 상자, 무지개 상자 이벤트
- 획득 아이템 정보 표시 (장비, 잠재력 항아리, 설계도, 조각, 연금 항아리)
- 12종 아이템 표시 최적화, 초과 시 스크롤
- **다시하기 기능 제거** (티켓 배율 기능으로 대체)
- 클리어 결과 화면 (나가기 버튼)

### 4. 챌린지 레이드 모드 로비
- 챌린지 보스 정보 패널
- 보상 정보 및 상세 보상 팝업
- 매수 설정 팝업 (1회 최대 3장, 매수 구매 시 최대 5/6장)
- 난이도별 클리어 기본/연합 보상
- Toast 메시지 시스템 (이전 난이도 미클리어 등)

### 5. 시스템 변경사항
- **일반 레이드 랭킹 기능 제거**
- 입장 전투력 조건 확인
- 단계별 수치 설정 (2000/3000 ~ 280,000)
- 티켓 압축 (5장 → 2장)
