# 쿠키 성장 가이드 TC 설계 계획

### 기능 목록

#### A. 쿠키 성장 가이드 (메인 기능)

**A-1. 진입**
- 쿠키 상세 화면 "공략" 버튼 노출 (쿠키 경험치 튜토리얼 진행 이후 활성화)
- 공략 버튼 터치 시 쿠키 성장 가이드 화면으로 이동
- 미보유 쿠키일 경우 기존 미보유 쿠키 정보 표시
- 쿠키 페이스 아이콘 표시 기준 (내 쿠키 화면 / 모드 별 쿠키 세팅 화면)

**A-2. 쿠키 가이드 화면 구성**
- 좌측 쿠키 리스트 영역 (가이드 쿠키 변경 가능)
- 우측 4종 카테고리 탭 (쿠키 / 장비 / 아티팩트 / 시즈나이트)
- 쿠키 변경 시 뒤로가기 동작 (처음 선택한 쿠키 상세보기로 복귀)

**A-3. 쿠키 카테고리**
- 쿠키 레벨업 (현재 / 최대 레벨 + 부족 재화 + 바로가기)
- 쿠키 성급업 (현재 / 최대 성급 + 부족 영혼석 + 바로가기, 재료 부족 라벨)
- 쿠키 스킬 레벨업 (60LV 해금 / 미달 시 토스트, 부족 재화 + 바로가기)
- 부족한 육성 자원 (펼치기/접기, 재화 종류 및 수량 표시, 획득하러가기 숏컷)
- 모든 육성 재화 획득 시 "모든 육성 재화를 획득했습니다." 표시

**A-4. 장비 카테고리**
- 현재 장착중인 장비 정보 (장비 아이콘/레벨/성급/잠재력 속성)
- 추천 장비 미장착 부위 경고 아이콘
- 인포(i) 버튼 → 추천 장비 팝업
- 일괄 장착하기 버튼
- 미보유 추천 장비 (장비 이름 + 아이콘, 획득하러 가기 숏컷)
- 추천 장비 상태 분기 (미획득/획득-미장착/장착완료)
- 즉시 장착하기 버튼 동작
- 상태별 리스트 처리 (미획득→획득 가기 / 미장착→장착하러 가기 / 장착→삭제)

**A-5. 아티팩트 카테고리**
- 현재 장착중인 아티팩트 정보 표시
- 미보유 추천 아티팩트 정보 표시
- 즉시 장착하기 버튼
- 아티팩트 별 포커싱 탭 분기 (픽업 레전더리/픽업 에픽/픽업 지난 레전더리/픽업 지난 에픽 3개월±)

**A-6. 시즈나이트 카테고리**
- 시즈나이트 미해금 시 탭 잠금 처리 + "탑레벨 N에 해금됩니다" 토스트
- 현재 장착중인 시즈나이트 정보
- 미보유 추천 시즈나이트 (이름 + 아이콘, 획득 숏컷)
- 추천 시즈나이트가 여러 종이라 장착 완료 후에도 리스트 잔존 가능

**A-7. 숏컷 기능 (각 카테고리별 진입 화면)**
- 쿠키 상세보기 화면 (쿠키 정보 탭 / 스킬 탭 / 장비 탭 / 시즈나이트 탭 포커싱)
- 쿠키 강화 화면 (레벨업 / 스킬 레벨업 / 승급)
- 장비 장착 화면 (무기/모자/상의/하의 별 포커싱)
- 아티팩트 (아티팩트 장착 / 승급 화면)
- 시즈나이트 (시즈나이트 장착 화면)

**A-8. 쿠키 레벨업 초기화**
- 쿠키 레벨업 화면 초기화 정보 상시 노출
- 쿠키 상세화면 [레벨업 초기화] 버튼 추가
- 2LV 이상 활성화 / 1LV 딤드 처리 (딤버튼 터치 시 토스트)
- 레벨 초기화 팝업: 쿠키 정보 / 초기화 정보 / 버튼 영역(초기화·취소)
- 초기화 진행 시 LV.1로 변경 + 모든 재화 100% 회수
- 스킬 레벨업 된 쿠키 초기화 플로우 (스킬 레벨 유지, 스탯 변화)
- 스킬 레벨업 버튼 → 레벨업 버튼 롤백
- 스킬 화면 [스킬 레벨업] 버튼 딤드 처리 + "쿠키 레벨 60달성 시 가능합니다" 토스트
- 60LV 재달성 전까지 스킬 레벨업 불가

**A-9. 추천 잠재력**
- 장착중 장비(에픽 이상) → [추천 잠재력 보기] 버튼 표시
- 미장착 장비 → 기존과 동일
- 추천 잠재력 보기 버튼 터치 → 추천 장비 팝업 표시 (장비 추천 조합)
- 사용 통계 탭 (어비스 레이드/백신 스쿼드/유리미궁)

#### B. 패배 화면 쿠키 성장 가이드 연결

**B-1. 성장 가이드 버튼 노출/미노출**
- 노출: 스토리 모드 싱글 / 성장 던전 / 어비스 던전 / 그림자 토벌 / 탑의 균열 싱글 / 백신 스쿼드
- 미노출: 어드벤처 모드 / 길드전 / 멀티 콘텐츠(레이드·어비스 레이드·탑균 멀티·스토리 멀티) / 유리미궁

**B-2. 성장 가이드 팝업 동작**
- 패배 화면에 성장 가이드 버튼 터치 시 가이드 팝업 표시
- 쿠키 페이스 아이콘 터치 → 해당 쿠키 가이드 표시
- 성장 가이드 기능 그대로 연결
- 예외: 장비/아티팩트/시즈나이트 "즉시 장착" → "장착하러 가기" 변경
- 결과 화면에서는 갈아입기 불가 (장비 교체 화면 이동 필요)
- 체험/대여 쿠키는 가이드 미표시
- 대여쿠키 1 + 내쿠키 2 → 내쿠키 페이스 아이콘 2개만 표시
- 체험쿠키 1만 → 버튼 자체 노출 안됨

### 중점 확인 사항

1. **재화 회수 정확성 (Critical 우선순위)**
   - 쿠키 레벨업 초기화 시 사용한 모든 재화 (코인 / 경험치 물약 소·중·대 / 돌파석)가 정확히 100% 회수되는지
   - 스킬 레벨업 한 쿠키 초기화 시 스킬 레벨 유지, 레벨만 1로 변경되는 분기 정확성

2. **상태 분기 처리 (장비·아티팩트·시즈나이트 공통)**
   - 미획득 → "획득하러 가기" 표시
   - 획득했으나 미장착 → "장착하러 가기" 표시
   - 장착 완료 → 리스트 삭제
   - 시즈나이트는 여러 종일 수 있어 장착 후에도 리스트 잔존 케이스 분기 검증

3. **해금/잠금 조건**
   - 공략 버튼: 쿠키 경험치 튜토리얼 진행 이후 활성화
   - 스킬 레벨업: 쿠키 레벨 60 달성 조건 + 토스트 메세지
   - 시즈나이트 탭: 타워레벨 N 해금 조건 + 토스트 메세지
   - 추천 잠재력 보기 버튼: 에픽 이상 장착 장비만 표시

4. **숏컷 포커싱 정확성**
   - 각 카테고리/탭별 정확한 포커싱 (쿠키 정보/스킬/장비/시즈나이트 탭, 무기/모자/상의/하의 부위)
   - 부족 재화 획득하러 가기 → 정확한 콘텐츠 로비로 진입
   - 다음 단계 오픈 시 경고 팝업, 소탕 불가 단계 시 로비까지만 보내주는 분기

5. **패배 화면 가이드 노출 조건 (콘텐츠별 분기)**
   - 노출 6종 (스토리 모드 싱글, 성장 던전, 어비스 던전, 그림자 토벌, 탑의 균열 싱글, 백신 스쿼드) 정확히 노출
   - 미노출 콘텐츠는 표시되지 않음
   - 체험/대여 쿠키 조합별 페이스 아이콘 표시 규칙
   - 결과 화면 즉시 장착 불가 → "장착하러 가기" 변경 동작

6. **알림/노티 시스템**
   - 카테고리 탭 그린닷/레드닷 (육성 가능 표시) 노출/제거 조건
   - 부족 재화 펼치기/접기 상태 유지
   - 부족 재화 없을 시 빈 상태 표시 처리

7. **토스트 메세지 (Normal 다수)**
   - 1LV 쿠키 초기화 시도
