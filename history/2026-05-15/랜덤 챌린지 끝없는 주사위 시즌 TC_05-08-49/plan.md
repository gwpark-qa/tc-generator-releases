### 기능 목록

**1. 시즌 진입 및 튜토리얼**
- 시즌 최초 진입 시 스페셜 이벤트 안내 팝업
- 메인화면 진입 시 주사위 박스 떨어지는 연출
- 2단계 미션 최초 진입 시 가이드 애니메이션 1회 재생
- 튜토리얼 1회 재생 후 재진입 시 미재생 처리
- 주사위 굴리는 연출과 드래그 애니메이션 순서 처리

**2. 주사위 획득 플로우**
- 축복 선택 화면 (3종 중 1개) - 캐릭터 선택 시 주사위 동반 획득 (최대 2개)
- 크리스탈/프리미엄 결제 분기 (4,000 / 2,275)
- 주자 선택 시 조합별 주사위 할당 (1주차/첫 주자 도전 버튼 미노출)
- 주사위 도전 버튼 말풍선(?) 노출/제거
- 어이달리기 플레이 보상 (최고 점수 기준 산정, 재도전 시 점수 미달이면 변동 없음)

**3. 주사위 미션 - 조작**
- 인포 버튼 터치 시 미션 목록 확인
- 미션 6단계 이후 무작위 미션 표기
- 주사위 드래그하여 칸에 등록 (진동/달그락 소리)
- 마이너스 버튼 터치하여 회수 (다른 소리)
- 주사위 상태 애니메이션 (Main_In/Rolling_Result/Slot_In/Grab/Hold)

**4. 주사위 미션 - 케이스**
- CASE1: 같은 눈 2개 등록 (가이드 반투명 표시)
- CASE2: 연속되는 주사위 등록 (순서 무관)
- CASE3: 4개 합계 22 초과 (마지막 슬롯에서만 유효성 체크)
- 조건 충족 시 초록(87EF00) / 미충족 시 빨강(FF4940) 색상 표시
- 잘못된 주사위 드래그 시 보관함 복귀 + 토스트
- 조건 충족 시 완료 버튼 활성화 + 사용 주사위 소멸 안내 말풍선

**5. 미션 완료 및 보상**
- 1~5단계 완료 시 보상 분기 (주사위/버프/버프3택1)
- 6단계 완료 시 보상 분기 + 무작위 미션 디폴트 진입
- CASE1: 목록 보상 모두 지급
- CASE2: 선택 보상 팝업

**6. 무작위 미션**
- 무작위 미션 디폴트 화면
- 인포 버튼 터치 시 무작위 미션 정보 팝업 (현재 미션 최상단/녹색)
- 미션 셀 케이스 (현재/텍스트형/완료/fixedSlot)
- 다음 미션 자동 포커싱 (미션 완료 팝업)
- ChangeRandomMission 타임라인 재생

**7. 특수주사위 / 미션변경 리롤**
- 특수주사위 팝업 (보유 주사위 그리드 + 도움말)
- 미션변경 리롤 (미션 창 반짝 연출, 횟수 차감)
- 리롤 횟수 0 시 "남은 변경 횟수가 없습니다." 토스트
- MissionReroll 타임라인 재생

**8. 주사위 합성 - 화면**
- 합성 팝업 (타이틀/X버튼/카테고리 탭 강조)
- 보유한 주사위 영역 (0/50 카운트)
- 정렬 드롭다운 (낮은순/높은순/최신순)
- 좌측 일반/특수/특수 주사위 보관함 탭

**9. 주사위 합성 - 다시 굴리기**
- 슬롯 1칸, 잔여 횟수 표기 (예: 4/5)
- 주사위 드래그 → 통 연출 시퀀스 (올라옴→흔들흔들+연기→꾸욱→펑+파티클)
- 자동스킵 옵션 ON 시 연출 생략
- "다시 굴리기 완료!" 결과 팝업 + 보유 주사위 인벤토리 추가
- 횟수 0 시 버튼 터치 토스트 (단, 슬롯 장착은 가능)

**10. 주사위 합성 - 눈+1 / 특수**
- 눈+1: 동일 눈 2개 슬롯, 반투명 가이드, 합성 성공 시 +1 주사위 획득
- 6눈 +1 시도 시 토스트
- 특수 합성: 합계 10 이상 시 특수주사위 랜덤 획득
- 특수주사위 합성 재료 사용 불가 토스트 (단, 리롤은 가능)
- 빈 슬롯 합성 시도 토스트
- 미션창 이동 후 복귀 시 슬롯 주사위 자동 회수
- 조건 미충족 드롭 시 보관함 복귀 + 토스트

- 눈+1 합성 시 +1 주사위 획득 및 재료 소비
- 특수 주사위 합성 결과 획득
- 미션변경 리롤 시 미션 변경 처리

### 중점 확인 사항

1. **튜토리얼 1회성 검증**: 최초 진입 시 1회만 재생되며 재진입/재랜딩 시 미재생되는지 (시즌 안내, 2단계 가이드 애니메이션 각각)

2. **조건부 도전 버튼 노출 분기**: 1주차 / 첫 주자 / 주사위 획득 후 말풍선 / 미션 진입 후 말풍선 제거 — 각 시점별 UI 상태 분리 검증

3. **재도전 점수 산정 로직**: 어이달리기 재도전 시 이전 최고 점수보다 낮으면 주사위 수량 변동 없음 (재화 보호 로직)

4. **유효성 체크 시점**: 합계 미션은 **마지막 슬롯에서만** 색상(FF4940/87EF00) 변경 발생 — 중간 슬롯 등록 시 색상 변경 미발생 확인

5. **잘못된 주사위 드래그 처리**: 보관함 자동 복귀 + 토스트가 **동시에** 발생 (미션/합성 양쪽 모두)

6. **단계별 완료 분기**: 1~5단계 vs 6단계의 완료 후 흐름 차이 (6단계는 무작위 미션 디폴트 진입)

7. **보상 케이스 분기**: CASE1(전체 지급) vs CASE2(선택 팝업) 각각의 분기 처리

8. **리롤 횟수 0 처리**: "남은 변경 횟수가 없습니다." 토스트 출력 / 합성 다시 굴리기는 횟수 0이어도 슬롯 장착은 가능

9. **자동 스킵 옵션 분기**: 통 연출 5단계 시퀀스(올라옴→흔들+연기→꾸욱→펑+파티클→결과) ON/OFF 시 표시 여부

10. **합성 재료 제약**: 6눈 +1 시도 차단 / 특수주사위 재료 사용 불가 (단 리롤은 가능)

11. **합성 슬롯 자동 회수**: 합성 슬롯에 주사위 장착 상태에서 미션 탭으로 이동 후 복귀 시 자동 회수 (데이터 손실 인식 우려 케이스)

12. **색상 코드 정확성**: 불충족 `FF4940`, 충족 `87EF00` 픽셀 단위 검증

13. **연출 순서 충돌 처리**: 주사위 굴리는 연출이 먼저 발생하는 경우 → 연출 종료 후 드래그 애니메이션 발생

14. **무작위 미션 정렬**: 현재 진행 미션이 최상단 + 녹색 표기

15. **주사위 종류별 등장 제약**: 통 안에는 종류당 1개씩만 등장 (보유 수와 무관)
