### 기능 목록

**1. 이벤트 진입/종료 처리**
- 이벤트 시작 시 모드리스트 노출 (NEW 배지)
- 이벤트 종료 시 모드리스트 제거
- 이벤트 종료 시점 대기/매칭/인게임 상태별 처리
- 이벤트 종료 30분 전 매칭 lock
- 미수령 보상 우편 지급
- 이벤트 종료까지 남은 시간 2단위 표기 (1일 이하 시 빨간색)

**2. 모드리스트**
- 이벤트 탭 하위 노출
- 신규 모드 느낌표닷
- 보상 획득 가능 모드 레드닷
- 상자 슬롯 진행도 표기 (없음/진행중/오픈 가능)
- 주간 미션 진행도/남은 시간 표기

**3. 플레이 로비 (메인 허브)**
- 이벤트 타이틀 표기
- 시즌 정보/서브타이틀
- 플레이 마감까지 남은 시간 (2단위)
- 유저 정보 영역 (칭호/길드/닉네임/쿠키 모델링)
- 유저 상태 표시 (준비중/READY/쿠키 설정중)
- 가이드북/랭킹/도감/채팅 버튼
- PLAY 버튼 (매칭 전/중/완료)
- 매칭 진행 시간/취소 버튼
- 파티 초대 (친구/길드/최근 함께한 유저/ID 참가)
- 파티 인원 변경 시 레디 취소
- 3인 파티 매칭 스킵

**4. 쿠키 선택**
- 쿠키 리스트 화면 진입
- 좌측 쿠키 모델링/타입/성급/레벨
- 자세히 보기 버튼/팝업
- 우측 쿠키 리스트 (스틸 버튼/성급/레벨/타입)
- 무작위 픽 (최상단 정렬, 히든 쿠키 안내)
- 다른 유저 선택 쿠키 제외
- 미보유 쿠키 0% 확률 처리
- 선택 완료 후 공개

**5. 유실물 센터 (보상 상자)**
- 슬롯 상태 (수령 가능/진행중/빈 슬롯)
- 빈 슬롯 노란닷
- 수령 가능 슬롯 레드닷
- 상자 선택 팝업 (3종 상자, 보유 개수, 미보유 표기)
- 상자 상세 확률 팝업 (보상 리스트 확률/대체 보상 확률)
- 시간 단축권 사용 팝업 (토글바, MAX 버튼)
- 슬롯 장착 후 해제 불가
- 상자 오픈 연출 (페이드 아웃 → 보상 노출)
- 푸시 알람 발송
- Last One 보상
- 대체 보상 지급 (모두 소진 후)

**6. 유실물 게시판 (보상 리스트)**
- 보상 리스트 정렬 (레어리티/개수 내림차순)
- 획득량 프로그레스
- 마일스톤 보상 (좌우 스크롤)
- 수령 가능 보상 레드닷/모두 받기
- 수령 완료 시 완료 마크+딤드
- 아이템 툴팁
- 대체 보상 리스트

**7. 주간 스탬프 (주간 미션)**
- 주간 미션 진행도/남은 시간 표기
- 레드닷 알림
- 스탬프 진행도 UI (미완료 딤드/완료 표시)
- 주간 미션 팝업
- 매주 수요일 9:00 초기화

**8. 도감**
- 쿠키 패널/드래프트 패널 탭 구분
- 쿠키 해금 상태 (해금/잠금/선택됨/미션 달성)
- 잠금 해제 시 활성화 + 느낌표닷
- 잠금 쿠키 회색/잠금 아이콘/해금 조건 표기
- 드래프트 패널 (사용함/사용 안 함/히든)
- 도감 달성도 팝업
- 쿠키 미션 달성도/드래프트 미션 달성도
- 신규 항목 추가 시 토스트
- 시즌 변경 시 초기화

**9. 가이드북**
- 이벤트 타이틀 모드 탭
- 라운드 규칙 탭
- 보상 상자 탭

**10. 랭킹**
- 리더보드 탭/랭킹 보상 탭
- 상위 1000위 노출
- 본인 100위 이상 시 "???" 표기
- 상위 10위 노란색 강조
- 본인 초록색 강조
- 무기록자 참여 독려 텍스트
- 등급 보상 (C/B/A/S/S+/L/N)
- 랭크 최초 달성 시 보상 지급

**11. 인게임 로딩창**
- 유저 카드 (칭호/닉네임/길드/쿠키)
- 무작위 픽 시 선택된 쿠키 공개
- 로딩 문구/로딩 바

**12. 결과창 (RESULT)**
- 점수/New Record/Grade 표시
- 플레이 정보 (별사탕/처치/소생)
- 드래프트 사용 목록
- 쿠키 영역 (본인 중앙, 본인 초록색/타인 흰색)
- Disconnected 표기
- 다시하기 (제한시간/회색 비활성화)
- 나가기
- 이모지/신고하기/유저 상태(체크/X)
- 함께한 유저 다시 초대 팝업
- "오늘 하루동안 보지 않기"

**13. 튜토리얼**
- 밀키웨이 가이드 멘트
- 유실물 상자 보유 후 트랜짓 입장 시 발동

**14. 예외 처리**
- 개별/전원 발주/디스커넥션
- 중단하기 투표 (3명/2명/1명)
- 매칭 중 이벤트 종료
- 인게임 중 이벤트 종료

### 중점 확인 사항

1. **이벤트 시간 경계 처리**
   - 이벤트 종료 30분 전 매칭 lock, 종료 시점 대기/매칭/인게임 상태별 분기 처리가 명확하게 구분되도록 작성
   - 이벤트 종료 1일 이하 시 타임바 색상 변경, 토스트 메시지 출력

2. **유실물 슬롯 상태 분기**
   - 빈 슬롯/진행 중/수령 가능 3가지 상태별 UI(노란닷·레드닷·배경) 및 상호작용 구분
   - 슬롯 장착 후 해제 불가, 시간 단축권 사용 시 필요 수량 초과 제한·MAX 동작 등 정밀한 조건 분기

3. **매칭 플로우**
   - 매칭 전/중/완료/취소/매칭 lock의 5단계, 3인 파티 시 매칭 스킵
   - 파티원 변경 시 레디 자동 취소, ID 참가/거절/가득 참 등 파티 초대 예외

4. **쿠키 선택 - 무작위 픽**
   - 다른 유저가 선택한 쿠키 제외, 미보유 쿠키 낮은 확률 처리, 모든 유저 선택 완료 후 공개 시점
   - 무작위 픽 시 로딩 화면에서 쿠키 공개되는 흐름

5. **도감 상태 분기**
   - 쿠키/드래프트 패널의 잠금/해제/선택/미션 달성/히든 5가지 상태별 UI 표시
   - 신규 항목 추가 시 토스트, 시즌 초기화 동작

6. **결과창 다시하기**
   - 제한시간 내 다시하기 시 전원/일부 차이에 따른 로딩창 분기, 제한시간 종료 시 비활성화
   - 함께한 유저 다시 초대 팝업 노출 타이밍 (나가기 → 로비 이동 후) 및 "오늘 하루동안 보지 않기" 동작

7. **네트워크/중단 예외**
   - 개별/전원 디스커넥션 시 보상·기록 처리, Disconnected 표시
   - 중단하기 투표 인원별 동의 조건 (3·2명/1명)

8. **알림 시스템**
   - 모드리스트의 느낌표닷/레드닷/NEW 배지/상자 슬롯 상태 표기가 모드 진입 전부터 확인 가능한지
   - 쿠키 도감 정보 미확인 시 느낌표닷, 확인 후 제거

9. **카테고리 통합**
   - 플레이 로비에서 진입하는 가이드북/랭킹/도감/유실물 게시판 등은 별도 main으로 분리하지 않고 적절한 main 하위로 통합 배치 (단, 유실물 센터는 슬롯/팝업이 다양해 별도 main으로 분리)
   - 화면 이동 케이스는 각 카테고리 끝, 예외 케이스는 최하위 카테고리 끝에 배치
