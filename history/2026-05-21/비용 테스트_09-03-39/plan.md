### 기능 목록

**1. 수면 시스템 (CC기)**
- 수면 걸림 조건 (오르골 작동 / 달빛술사 쿠키 공격)
- 수면 적용 대상 (중간보스/보스 제외 모든 몬스터)
- 수면 상태 처리 (스턴과 동일, 스킬 캔슬, BT 초기화)
- 수면 전용 애니메이션 (Sleep Parameter)
- Fly 몬스터 추락 처리
- 수면 해제 조건 (시간 경과 N초 / 피격)
- 틱뎀(DOT) 예외 처리
- 수면 해제 후 달빛술사 피해 증가 디버프
- 수면 이펙트 (머리 위 표시 / 동시 다수 적용)

**2. 오르골 기믹**
- 상태 머신 (비활성화 → 활성화 → 작동 → 비활성화)
- 활성화 범위 (반지름 10.5m 실선 표시)
- 게이지 차오름 (약 6초)
- 영역 내 적 사망 시 게이지 가속 (0.5초 단축)
- 범위 이탈 시 게이지 감소 (Delay 후 시작)
- 작동 효과 (Enemy 수면 디버프 부여)
- 비활성화 대기 시간 ({N}초, GmTile별 설정)
- 애니메이션 (활성화/작동/비활성화)
- 이펙트 (실선/차오름 색상/수면 퍼짐)

**3. 수면 보호막**
- 부여자 최대 체력 비례 보호막 양
- 수면 피격 시 즉시 소멸
- 일반 피격 시 소멸
- 중첩 불가
- 중간보스 이상 미사용
- 전용 UI (색상 차별화)

**4. 악몽 요정 (일반 몬스터)**
- 패트롤 대기 (인식 거리 20m)
- 플레이어 발견 시 "!" 표시
- 이동속도 700
- Sting 스킬 (12m 시작, 1.5m 데미지)
- Stagger로 공격 중단 시 _Skill_A_03 → _Skill_A_01 복귀
- 배치 수 4~6마리

**5. 악몽의 아가양 (일반 몬스터)**
- 패트롤 대기
- 플레이어 발견 시 "!" 표시
- 이동속도 600
- Rush 스킬 (15m 시작)
- 가속도 Curve (2초)
- 90도 이상 벗어남 시 정지
- 낙사 지형 앞 즉시 정지
- 충돌 성공/실패 애니 분기
- 충돌 시 플레이어 뒤로 밀림
- 공격 중 경직 면역
- 배치 수 1~3마리

**6. 노래하는 인형 (일반 몬스터)**
- 패트롤 대기 (몽롱하게 졸린 듯)
- 가장 가까운 몬스터 4m 거리까지 이동
- 이동속도 500~600
- Skill_A (3초간 노래, 반지름 9m, 3회 보호막 부여)
- 보호막 받은 몬스터 TargetList 제외
- 수면 진입 시 엎어져 잠
- 기상 시 3초 소요
- 도망 (주변 몬스터 0 + 2초 후, 3초 소요 Despawn)
- Enemy remains 카운트 O / 수집품 카운트 X
- 배치 수 1~2마리

**7. 악몽 램프 (일반 몬스터)**
- 고정형 (이동 X)
- 랜덤 방향 대기
- Skill_A (Area Attack, 십자 패턴, 인디케이터 2초)
- 경우에 따라 45도 회전(X자) 분기
- 경직 면역
- 배치 수 1마리 (경우에 따라 2마리, 1초 텀)

**8. 꿈꾸는 모빌 (엘리트 몬스터)**
- 패트롤 대기 → 플레이어 발견 시 소환 시작
- 이동속도 500~600
- Skill_A (별 투사체, 4.5~9m, 최대 3마리 소환)
- 소환 몬스터 랜덤 (아가양/악몽요정/악몽램프)
- Skill_B (모여드는 항성, 3m 범위, 5m 영역, 3.5초 대기 후 흡인)
- 수면 시 추락 + 추락 데미지
- 수면 시 소환 몬스터 전부 사망
- 기상 시 3초간 공중 복귀
- 경직 면역
- 배치 수 1마리

**9. 악몽의 양 (중간보스)**
- 패트롤 대기
- 이동속도 600
- Skill_A (Rush, 무한 사거리, 스턴 부여, 최대 3연속)
- Skill_A 고난이도: 오브젝트 충돌 시 Skill_B 투사체 1회 낙하
- Skill_B (High Angle Fire + Add on, 12m, Flight 1초, 반지름 2m, 4m 튕김 2회 추가)
- Skill_C (Area Attack, 무한 사거리, 2초, 반지름 3.5m)
- Skill_C 낮은 체력 시 별똥별 갯수 증가
- 공격 중 경직 면역
- 배치 수 1~3마리

**10. 꿈을 쏟아내는 모빌 (중간보스)**
- 공중 부유 (지상 7m, Hide 상태)
- 맵 주변 선회 이동
- Skill_A (3개 동시 투사체, 3m~12m, 매 3마리 소환)
- 소환 몬스터 2마리 이하 시 재소환
- Skill_A 소환 종류 (아가양/악몽요정/노래하는 인형)
- Skill_B (Area Attack, 15m, 더 자주 사용)
- 수면 시 추락 + 소환 몬스터 전부 사망
- 기상 후 5초간 지상 체류 (추가 타격 가능)
- 이후 0.5초간 공중 복귀

**11. 거대한 악몽 램프 (중간보스)**
- 고정형
- Skill_A (Summon + Area Attack Deliverers, 6m×6m 십자, 25 포인트 중 선정, 인디케이터 2초)
- Skill_A 예외: 가끔 쿠키 단독 위치 타격
- Skill_B (Melee, 반지름 8m, 120도 부채꼴, 인디케이터 1.8초)
- Skill_C (Laser, 3~15m 직선, 인디케이터 1초, 0.5초 발사)
- 1페이즈 / 2페이즈 분기 (체력 일정 이하 시)
- 2페이즈 카메라 후방 고정, 정해진 패턴 공격

**12. 어비스레이드: 악몽A (보스)**
- 맵 구성 (66m×54m 직사각형)
- 1페이즈 (낙사 불가)
- 2페이즈 진입 (별의 분노로 맵 가장자리 소멸, 낙사 가능)
- 낙사 시 안개 붙잡힘 (체력 비례 20~30% 피해)
- 붙잡힌 상태 매초 5~10% 피해
- 버튼 연타 탈출 (누적 시 횟수 증가)
- 악몽의 허상 기믹 (시각만, 데미지 X)
- 악몽 방출 (관통 후 고정)
- 악몽 회수 (경로 피해)
- 심연의 손길 (1차 속박, 2차 폭발)
- 진짜 찾기 (분신/본체 구분, A/B 캐스팅)
- 악몽 폭주 (흡수 악몽 수 비례 데미지)
- 잠식 (덩어리 낙하 → 웅덩이, 강화/강화2 분기)
- 악몽 정화 필살기 (9단계 흐름, 파괴/무력화/상쇄 분신, 성공 시 그로기/실패 시 전투 지속)
- 이동기 (정해진 방향 고속 이동)
- 텔레포트 (전조 후 순간이동)
