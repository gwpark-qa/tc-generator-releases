### 기능 목록

**1. 튜토리얼**
- 최초 진입 컷씬 (일카페이 쿠키 안내 방송 / 17단계 대화)
- 쿠키 선택 안내, 매칭 안내, 드래프트 안내, 유실물 안내
- 유실물 박스 정화 안내, 시간 단축권 사용 안내
- 튜토리얼 스킵/완료 후 재진입 미노출
- 유실물 보유 + 트랜짓 조건 충족 시 단축권 튜토리얼 트리거

**2. 이벤트 진입/종료 처리**
- starttime 도래 시 모드리스트 노출
- endtime 도래 시 모드리스트 제거 및 강제 이동(매칭 중/플레이 중)
- 종료 30분 전 매칭 lock
- 미수령 보상 우편 지급

**3. 모드리스트 (이벤트 카드)**
- 카드 상태 분기(최초 진입 / 배치 가능 / 여는 중 / 오픈 가능 / 잠김)
- 이벤트 타이틀·서브타이틀·잔여 시간·랭크 보상 배지
- 주간 퀘스트 진행도 도트, 인포 버튼
- 이벤트 로비 진입

**4. 이벤트 로비 (메인 화면)**
- 상단 타이틀/잔여 시간/인포/뒤로가기
- 탑바(재화, 2주년 코인, 메뉴, 홈), 메뉴 알림 닷
- 캐릭터 디오라마(3D)
- 좌측 메뉴(가이드북 / 랭킹 / 도감) 진입
- 이벤트 상점 바로가기
- 최고 기록 영역(등급 배지 / 점수 / 기록 없음 / 인포 → 랭킹 이동)
- 유저 상태(기본 / 쿠키 세팅 중 / READY)
- PLAY 버튼 상태 전환(기본 / 매칭 중 / 매칭 완료 / 취소)
- 길드/칭호 노출 정책

**5. 가이드북 / 인포 팝업**
- 이벤트 타이틀 모드 탭, 라운드 규칙 탭, 보상 상자 탭
- 인포 팝업(번역키)

**6. 쿠키 선택**
- 쿠키 그리드(해금 / 미해금 / 히든), 무작위 픽 셀
- 5성 60레벨 고정 안내
- 쿠키 미션 영역(왕관 진행 / 미션 리스트 / Shiny 이펙트)
- 자세히 보기 팝업 (UIPopup_CookieSkillInfo): 스킬 5종, 속성 표식/디버프 태그
- 쿠키 승급 정보 진입
- 무작위 픽 처리(다른 유저 미선택 / 히든 등장)
- 선택 미완료 만료 / 동시 선택 / 시간 만료 처리

**7. 매칭 시스템**
- 인원수별 매칭(1/2/3인 풀파티 스킵)
- 시간 단계별 중복 허용도(30/60/90/120초)
- 매칭 취소 / 네트워크 끊김

**8. 인게임 진입 로딩**
- 유저 카드(칭호 / 닉네임 / 길드명 / 쿠키), 무작위 ??? 표기
- 로딩 문구, 로딩 바

**9. 결과 화면 (RESULT)**
- 타이틀/라운드/플레이 타임/점수/Grade/New Record
- 쿠키 배치(본인 중앙), 닉네임 색상, Disconnected
- 플레이 정보(별석탄/몬스터/파티원 살린 수, 본인 최상단)
- 드래프트 정보(메인 3 + 서브 별 강화)
- 보상 영역(아이콘/카운트/툴팁)
- 하단 멀티(이모지 / 신고 / 유저 상태)
- 다시하기(카운트다운 / 비활성 / 모두 다시하기 / 일부만)
- 다시 초대하기 팝업(친구 추가 / 다시 초대 / 오늘 하루 보지 않기)
- 나가기

**10. 도감**
- 로비/탭 구조(쿠키 / 드래프트)
- 쿠키 도감(해금 / 미해금 / 히든 / 정보 패널 / 자세히 보기)
- 드래프트 도감(카테고리 필터 / 사용 경험 / 미사용 ???)
- 도감 달성도 팝업(쿠키 미션 / 드래프트 수집, 보상 일괄 수령, 자동 포커싱)

**11. 랭킹**
- 랭킹 리스트 탭(페이지네이션 / 본인 강조 / 기록 없음)
- 랭킹 정보(보상) 탭(등급별 보상 / 현재 등급 강조 / 즉시 지급)
- 본인 표기 규칙(1000위 이내 / 1000위 이후 % / 기록 없음 "-")

**12. 유실물 슬롯 (피냐타 정화)**
- 슬롯 상태(NoItem / Empty / InProgress / Complete)
- VFX(InProgress / Complete) 분리, 뉴 닷/레드 닷
- 피냐타 선택 팝업(보유 리스트 / 보상 리스트 탭 / 대체 보상 탭)
- 슬롯 배치(자동 최고 등급 / 미보유 토스트)
- 정화 진행(타이머 / 슬롯 해제 차단 / 푸시 알림)
- 시간 단축권 사용 팝업(슬라이더 / MAX / 자동 보정 / 0개 비활성)
- 정화 완료 보상(SlotOpen 연출 / 미확인 추출 / 확률 실시간 변경 / 대체 보상)
- Last One 보상, 보너스 보상
- 시즌 종료 시 삭제, 디바이스 시간 어뷰징 차단

**13. 주간 유실물 미션**
- 로비 진행도(진행 중 / 보상 가능 / 완료)
- 유실물 게시판 팝업(리스트 / 단계별 보상 / 대체 보상 안내)

**14. 주간 스탬프 미션**
- 로비 스탬프 패널 5가지 상태
- 주간 미션 팝업(초기화 시간 / 미션 리스트 / 완료 보너스 / KST 수요일 9시 초기화)

### 중점 확인 사항

**1. 인게임 케이스 제외 기준 엄수**
- 일시정지/중단 투표, 부활 시스템, 라운드 진행 등 인게임 내부 동작은 작성 제외.
- "PLAY 버튼 터치 시 매칭/로딩 진입"까지만 작성하고 그 이후 인게임 플레이는 미작성.
- 단, RESULT(결과 화면)은 아웃게임 UI로 간주하여 포함.

**2. 슬롯 상태 4종 × 연출 분기 (피냐타 정화)**
- NoItem / Empty / InProgress / Complete 각각의 클릭 동작, VFX 표시 위치(InProgress 하위 / Complete 하위), 뉴 닷·레드 닷 동시 표시 우선순위까지 빠짐없이 분리.
- 시간 단축권 슬라이더의 +/-/MAX, 잔여 시간 자동 보정, 0개 비활성, 단축 후 0초 도달 시 동작까지 케이스화.

**3. 매칭 흐름의 단계적 조건 분기**
- 1/2/3인 인원수별 매칭 흐름(풀파티 스킵), 30/60/90/120초 단계별 중복 허용도 변경, 매칭 lock(종료 30분 전), 매칭 중 네트워크 끊김 → 카테고리별로 분리.
- 매칭 중 endtime 도래 시 강제 취소 + 토스트, 인게임 중 endtime 도래 시 플레이 마지막까지 진행 등 예외 처리 2종 구분.

**4. 결과 화면 UI 표기 정합성**
- 본인 최상단 정렬, 닉네임 색상(본인 초록/타 유저 흰색), Disconnected 표기, 쿠키 중복 픽 숫자 표기, New Record 조건 분기, 다시하기 카운트다운 만료 시 비활성, 모두/일부 다시하기 분기.

**5. 도감 상태 표현 (해금/미해금/히든)**
- 쿠키·드래프트 모두 3가지 상태가 존재. 미해금은 해금 조건 노출 vs 히든은 ??? 유지 정책 차이를 케이스로 명확히 분리.
- 미션 완료 왕관 Shiny, 신규 해금 느낌표돗트(탭 후 제거), 자세히 보기에서 미해금 쿠키도 정보 열람 가능 여부.

**6. 랭킹 표기 규칙 3종**
- 1000위 이내 N위 / 1000위 이후 상위 % / 기록 없음 "-" 표기 → 동일 UI 영역이지만 3개 케이스로 반드시 분리.
- 등급 컷 달성 시 즉시 지급 여부와 기록 없음 패널의 "랭킹에 도전하세요" 안내 분리.

**7. 카테고리 통합 정책 일관성**
- 가이드북/랭킹/도감/유실물 슬롯/주간 미션은 모두 이벤트 로비 진입 버튼을 가지지만, 각자 복잡한 내부 기능이 있으므로 별도 main으로 분리. 단, 로비 측 진입 버튼 TC는 "이벤트 로비 > 좌측 메뉴 패널" 하위 sub2에만 두고 화면 전환 케이스로 마무리 → 진입 후 기능은 각자 main에서 다룸.

**8. 예외 케이스 배치**
- 각 sub 카테고리 마지막에 네트워크 끊김/시간 임의 변경/이벤트 종료 임박 등 예외 처리를 배치.
- 화면 이동 케이스(예: 인포 버튼 → 랭킹 이동, 자세히 보기 → 승급 정보 이동)는 해당 sub의 가장 마지막에 배치.

**9. 단일 기대결과 원칙 준수**
- 토글/탭 전환 등 "여러 상태"가 있는 케이스는 각 상태별 TC로 분리.
- "다시하기 누름 / 나가기 누름"은 각각 별개 TC, "체크 마크 / X 마크 / Disconnected"는 sub3 단위로 분리.

**10. 튜토리얼 17단계 처리**
- 17단계 대사가 너무 세부적이므로, 컷씬/안내 단계별로 그룹화하여 단계별 진행과 스킵·완료 후 미노출만 TC화. 개별 대사 문구 검증은 비기능 케이스로 간주하여 제외.
