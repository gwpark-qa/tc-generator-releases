### 기능 목록

이미지와 텍스트를 교차 검증하여 파악한 기능 목록:

1. **상점 내 프리미엄 상품 노출**: 상점 화면에서 어비스 퀘스트 프리미엄 상품이 표시됨
2. **판매 조건**: 어비스 퀘스트 시즌 활성화 상태 + 프리미엄 패스 미구매 시에만 상품 표시
3. **상품 구매 팝업**: 상품 터치 시 구매 팝업 노출 (상품명, 보상 아이템 목록, 잔여 시간, 가격 표시)
4. **구매 완료 처리**: 구매 후 상점 내 상품이 "구매 완료" 상태로 변경
5. **구매 후 연출**: "어비스 퀘스트 프리미엄 보상 오픈" 팝업 연출
6. **어비스 퀘스트 화면 - 마일스톤 보상**: 누적 포인트 기반 마일스톤 구간별 무료/유료 보상 표시
7. **프리미엄 보상 마일스톤**: 무료 보상 하단에 프리미엄 보상 마일스톤 추가, 기존과 동일 포인트 구간
8. **프리미엄 패스 미구매 시 잠금 표시**: 프리미엄 보상 아이콘에 잠금 아이콘 표시
9. **잠금 보상 터치 시 상점 팝업 노출**: 잠금 상태의 보상 아이콘 터치 시 상점 상품 팝업 이동
10. **프리미엄 구매 배너**: 하단에 프리미엄 패스 구매 유도 배너 노출, 터치 시 상점 이동
11. **프리미엄 구매 완료 시 UI 변경**: 구매 완료 뱃지 표시, 잠금 해제, 배너 제거
12. **프리미엄 보상 수령**: 마일스톤 달성 시 프리미엄 보상 획득 가능
13. **시즌 종료 시 미획득 보상 메일 발송**: 유료 상품 구매 후 시즌 종료 시 미획득 보상을 메일로 제공
14. **어비스 퀘스트 시즌 타이머**: 시즌 종료까지 남은 시간 표시
15. **퀘스트 카테고리 탭**: 어비스 던전, 어비스 레이드, 잊혀진 기억의 제단, 시즈나이트 등 카테고리
16. **미션 목록 및 바로가기 버튼**: 각 미션별 조건, 진행도, 보상, 바로가기 버튼 표시
17. **정보 버튼(ⓘ)**: 어비스 퀘스트 타이틀 옆 정보 버튼

### 중점 확인 사항

1. **결제 관련 TC (Critical 집중)**: 상점에서 프리미엄 상품 구매 시 결제 처리, 구매 후 상품 상태 변경, 구매 후 프리미엄 보상 해금이 정상 연동되는지 — 결제 오류/중복 결제/미지급은 치명적
2. **프리미엄 보상 지급 정확성**: 프리미엄 패스 구매 전/후 보상 획득 가능 여부가 정확히 분기되는지, 무료 보상과 유료 보상의 구간 분리가 기획 의도대로 동작하는지
3. **시즌 종료 시 미획득 보상 메일 발송**: 유료 상품 구매 후 시즌이 종료될 때 미수령 프리미엄 보상이 메일로 정확히 발송되는지 (무료 보상은 해당 안 됨)
4. **미션 내용/상품 종류의 추상화**: 기획서 중점사항에 따라 변경 가능한 미션 내용, 상품 종류, 보상 아이템 등은 구체적 수치/이름을 명시하지 않고 일반화하여 작성
5. **잠금 보상 터치 → 상점 연동**: 미구매 상태에서 잠금 보상 터치 시 상점 상품 팝업으로 정확히 연동되는 크로스 화면 플로우
6. **판매 조건 분기**: 시즌 비활성화 시 상품 미노출, 이미 구매 완료 시 구매 완료 상태 표시 등 조건별 분기 검증
