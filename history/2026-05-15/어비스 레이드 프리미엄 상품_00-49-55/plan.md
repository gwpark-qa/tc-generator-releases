### 기능 목록

기획서 텍스트 및 이미지에서 파악한 기능은 다음과 같습니다:

1. **도전 미션 버튼** — 어비스 레이드 로비에 신규 배치된 진입 버튼 (어비스 레이드 관련 미션만 다루며, 다른 곳에는 표시되지 않음)
2. **도전 미션 팝업 표시** — 도전 미션 버튼 터치 시 미션 팝업이 출력됨
3. **미션 리스트 표시** — 현재 시즌에 해당하는 미션 리스트를 표시
4. **미션 조건 텍스트** — 각 미션의 조건 내용이 텍스트로 표시됨
5. **미션 조건 횟수** — 수행 횟수 / 필요 횟수가 프로그레스 바 형태로 표시됨 (예: 0/500,000)
6. **획득 보상 표시** — 보상 아이콘 및 획득 개수가 표시됨 (예: 보석 아이콘 + 100)
7. **받기(획득) 버튼 — 미달성 상태** — 조건 달성 전일 경우 딤드(비활성화) 처리
8. **받기(획득) 버튼 — 달성 상태** — 조건 달성 시 활성화되며, 터치 시 획득 가능한 모든 보상을 획득
9. **미션 리스트 정렬** — 보상 획득 가능한 미션 최상단 정렬, 보상 획득 완료한 미션 최하단 정렬
10. **팝업 닫기** — X(닫기) 버튼 터치 시 팝업 닫힘
11. **어비스 퀘스트와 마일스톤 비공유** — 도전 미션은 어비스 퀘스트와 마일스톤을 공유하지 않음
12. **즉시 유효 재화 지급** — 도전 미션 하나 클리어 시 즉시 유효 재화 지급 (마일스톤 없이 개별 보상)
13. **미션 리스트 스크롤** — 미션이 여러 개일 경우 스크롤하여 확인 가능 (이미지에서 4개 이상 미션이 나열되어 스크롤 가능한 UI 확인)

### 중점 확인 사항

1. **받기 버튼 상태 전환 정확성** — 미달성(딤드) → 달성(활성화) → 획득 완료 3단계 상태가 정확히 전환되는지, 특히 "획득 가능한 모든 보상을 획득한다"는 문구에서 다중 보상 일괄 지급 동작 검증 필요
2. **미션 리스트 정렬 로직** — 보상 획득 가능 미션이 최상단, 획득 완료 미션이 최하단으로 정렬되는 동적 정렬 로직을 다양한 상태 조합에서 검증
3. **어비스 퀘스트와의 독립성** — 마일스톤을 공유하지 않는다는 점, 도전 미션 보상이 어비스 퀘스트 진행도에 영향을 주지 않는지 확인
4. **즉시 유효 재화 지급** — 마일스톤 없이 개별 미션 클리어 시 즉시 재화가 지급되는 구조이므로, 재화 차감/증가가 정확한지 Critical 수준으로 검증
5. **시즌 연동** — "현재 시즌에 해당하는 미션 리스트" 표시이므로, 시즌 변경 시 리스트 갱신 여부 확인
6. **도전 미션 버튼 노출 범위** — 어비스 레이드 로비에만 표시되고 다른 곳에는 표시되지 않아야 함
7. **미션 조건 횟수 프로그레스** — 수행 횟수/필요 횟수가 실시간으로 반영되는지, 프로그레스 바와 수치가 일치하는지 확인
