# TC 설계 계획

### 기능 목록

기획서에서 파악한 주요 기능을 흐름 순서대로 정리합니다.

#### A. 레이드 모드 진입 화면
1. 로비 → 햄버거 메뉴 → 레이드 모드 진입 (전환)
2. 상단 뒤로가기(<) 버튼
3. 상단 "상시 / 이벤트" 토글 탭
4. 좌측 보스 진입 카드 리스트 (불타는 곳 레이드, 성장 던전, 탑의 군락, 어비스 던전 등)
5. 우측 메인 보스 카드 (어비스 일러스트/보스명/진입 정보)
6. 보스 카드 활성/잠금 상태 아이콘 표시
7. 잠긴 보스 터치 시 잠금 안내 팝업 출력
8. 보스 카드 선택 시 레이드 모드 로비로 전환

#### B. 레이드 모드 로비
1. 싱글 / 멀티 전환 토글 버튼 (Default: 싱글)
2. 싱글 모드 선택 시 동일 화면 UI 갱신
3. 멀티 모드 선택 시 멀티 매칭/파티 화면으로 전환
4. 보스 정보 표시 영역 (보스명, 초상화, 배경, 서브 설명)
5. 보스 정보 좌측 영역 터치 시 보스 정보 팝업 출력
6. 난이도 변경 버튼 (현재 난이도 표시)
7. 난이도 변경 버튼 클릭 시 변경 팝업 출력
8. 난이도 변경 팝업 재클릭 시 닫힘(토글) 동작
9. 보스 변경 위치 버튼 (보스 변경)
10. 난이도 단계 버튼 (1~10단계 / 잠금 표시)
11. 입장 전투력 표시
12. 추천 속성 표시
13. 적용 조건 표시
14. 입장 전투력 부족 시 난이도 비활성화 + Toast 출력 (`equipment.dungeon.not.host`)
15. 이전 난이도 미클리어 시 비활성화 + Toast (`single_boss_challenge.locked`)
16. 빠른 난이도 변경 버튼
17. 빠른 난이도 변경 팝업 출력
18. 빠른 난이도 변경 시 자격 미달 Toast 처리
19. 보상 정보 표시 영역
20. 보상 정보 터치 시 보상 상세 팝업
21. 보상 상세 팝업 내 보스 변경 좌우 버튼
22. 보상 상세 팝업 내 난이도별 보상 정보 표시
23. 클리어 기본 보상 / 연관 보상 표시 (드랍 확률 표시)
24. Play 버튼 (티켓 소모 표시)
25. 배수 설정 버튼 (1회 최대 3장 / 패스 시 최대 5장)
26. Play 버튼 비활성화 조건 + Toast 출력
27. 티켓 입수 변경 (5장 → 2장) 반영 표시
28. 멀티 모드 진입 시 매칭/파티 UI 추가 표시

#### C. 레이드 인게임 (UI 진입 관련만)
1. Play 버튼 터치 시 인게임 진입(로딩 화면)
2. (제거 검증) 코코방울 연출 미출력
3. (제거 검증) 코코방울 축복 포인트 영역 미출력
4. (제거 검증) 랜덤 상자 기능 미출력
5. (제거 검증) 트로피 포인트 영역 미출력
6. 보스 처치 후 아이템 드랍 형태 표시 (필드 드랍)
7. 드랍 연출 약 3초 표시
8. 클리어 화면(CLEAR) 진입
9. 획득 아이템 정보 표시 영역
10. 12종 아이템 최적화 표시 + 초과 시 스크롤
11. 아이템 클릭 시 툴팁 출력
12. 드랍 품목 종류별 표시 (장비/잠재력 항아리/설계도/조각/연금 항아리)
13. (제거 검증) 다시까기 기능 제거
14. 클리어 화면 내 트로피/축복/무지개 상자 영역 미표시
15. 나가기 버튼 터치 시 로비 복귀

#### D. 챌린지 레이드 모드 로비
1. 일반 모드 ↔ 챌린지 모드 탭 전환
2. 챌린지 레이드 보스 진열 표시
3. 보스 일러스트/보스명/보상 표시
4. 입장 정보 / PLAY 버튼
5. (제거 검증) 코코방울의 축복 영역 미표시
6. (제거 검증) 트로피 기능 미표시
7. (제거 검증) 무지개 상자 이벤트 미표시
8. (제거 검증) 일반 레이드 랭킹 진입 버튼 미표시
9. 챔피언 점수/단계 진행 표시
10. 챕터 보상 미션 아이콘 표시
11. 2인 멀티 플레이 진입 (4인 → 2인 변경 반영)
12. PLAY 버튼 터치 시 인게임 진입(로딩)
13. 클리어 화면 트로피 영역 빈 형태 표시
14. 클리어 화면 "시작하기"/"나가기" 버튼

#### E. 제작소 화면 (장비 연금 진입 메뉴 추가만 - 상세 시스템 제외)
1. 로비 → 햄버거 → 제작소 진입
2. 제작소 화면 메뉴 정렬 순서 (장비 제작 → 장비 연금 → 아티팩트 제작 → 아이템 제작)
3. 장비 연금 진입 메뉴 버튼 신규 표시
4. 장비 연금 버튼 터치 시 장비 연금 화면으로 전환

#### F. 레이드 모드 탑레벨 화면 (구성/전환만 - UI 변경 상세 제외)
1. 햄버거 → 성장 탭 → 레이드 모드 진입
2. 탑레벨 메뉴 트리 표시 (성장 던전 / 레이드 모드 / 탑의 균열 / 어비스 던전)
3. 레이드 모드 메뉴 선택 시 우측 보스 리스트 표시
4. 보스 진입 버튼(→) 터치 시 해당 보스 로비 전환

- **Critical**: 핵심 진입 차단/재화·티켓 손실 가능성/결제 관련 케이스 (예: Play 버튼 터치 시 인게임 진입, 티켓 소모 처리, 보스 카드 선택 시 로비 전환, 챌린지 PLAY 인게임 진입, 제작소 장비 연금 진입)

- **Normal**: 단순 UI 표시/연출/효과/정렬 순서/배경 이미지 표시

### 중점 확인 사항

1. **제외 항목 엄수**
   - 변경 플로우, 연금술 이벤트 삭제 자체, 장비 연금 시스템 상세, 연금 항아리 드랍 변경, 탑레벨 6종 보스 정보 표시 변경 상세, 데이터 테이블, 이펙트 요청 리스트는 TC 작성에서 제외
   - 단, 제작소 "장비 연금 메뉴 버튼 추가/전환"과 탑레벨 "화면 진입/구성/전환"은 포함

2. **제거 기능 검증 케이스 누락 방지**
   - 인게임/클리어/챌린지 로비에서 제거되는 항목들이 노출되지 않는지 검증하는 케이스를 별도 sub2/sub3로 명시
   - 코코방울 연출, 코코방울 축복 포인트, 트로피 시스템, 무지개 상자, 랜덤 상자, 다시까기, 일반 레이드 랭킹

3. **화면 전환 케이스 누락 방지**
   - 각 main 카테고리의 마지막에 화면 전환 TC 배치
   - 로비→레이드 진입, 진입→로비, 로비↔모드 토글, Play→인게임, 인게임→클리어→로비, 일반↔챌린지, 로비→제작소, 제작소→장비연금, 탑레벨→보스 로비

4. **수치/스펙 정확성**
   - 티켓 입수 5장→2장 변경
   - 1회 플레이 최대 티켓 소모: 기본 3장 / 패스 5장
   - 난이도 10단계, 보스 6종
   - 드랍 연출 약 3초
   - 클리어 아이템 최대 12종 최적화 + 스크롤
   - 멀티 4인 → 2인 변경

5. **Toast 메시지 케이스 분리**
   - `equipment.dungeon.not.host` (입장 전투력 부족) / `single_boss_challenge.locked` (이전 난이도 미클리어) — 각각 별도 TC

6. **싱글/멀티 토글 Default 상태**
   - Default: 싱글 / 클릭 시 모드 반영 — Default 상태 자체를 TC로 별도 검증

7. **인게임 내부 동작은 제외**
   - "Play 버튼 터치 시 로딩 화면 진입"까지만 작성하고, 그 이후 전투/플레이 케이스는 작성하지 않음
   - 단, 클리어 화면(보상 표시/나가기) 및 드랍 연출은 UI/UX 관련이므로 포함

8. **카테고리 sub3 적극 분화**
   - 단순히 "버튼 표시" 수준이 아니라 "팝업 출력 / 화면 전환 / 토글 닫힘 / Toast 출력 / 비활성화" 등 유저 상호작용 결과를 sub3로 명확히 분리
