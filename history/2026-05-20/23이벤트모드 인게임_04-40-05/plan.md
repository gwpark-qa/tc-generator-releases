- 열차 출발 연출
- 초기 UI 표시 (연료 게이지, 경험치 게이지, 시간, 스코어)
- 3인 멀티 플레이어 환경 구성

**2. 캐릭터/조작 시스템**

- 쿠키 카메라 영역

**4. 인게임 UI**

- 연료 0 도달 시 실패
- 경험치 게이지 (최대 100, 레벨업 시 초기화, 최대 15레벨)
- 레벨업 연출 및 서브 드래프트 등장
- 시간 표시 (좌상단)
- 스코어 UI (우상단)
- 위치 인디케이터 (보스/엘리트/약초 등)
- 라운드 진도 현황 UI
- 알림 텍스트 배너 ("적들이 몰려옵니다!")

- 재접속 가능
