# 쿠키 스토리 던전 - 1차 TC 설계 계획

### 기능 목록

#### A. 컨텐츠 진입 경로
- 메인 로비의 PLAY 버튼
- 모드리스트의 스토리 던전 모드 (티켓/최종 보상/진행중인 시즌 종료까지 남은시간/신규 시즌·이벤트 오픈 시 노출 레드닷)

#### B. 최근 책장 (Recent Bookshelf)
- 타이틀 영역 (황혼 도서관, 시즌 종료까지 남은 시간, 도움말 버튼)
- 책(챕터) 선택 영역 (오픈된 챕터 비활성→활성 연출, 잠금 챕터 비활성, 챌린지 모드 진입 조건/조건 미달 토스트)
- 시즌 목록 (보기 버튼, 목록 팝업, 지난 시즌의 진행도 표시, In Progress/Closed 구분)
- 코인 획득 가능 노티
- 교환소 버튼 (가장 최근 오픈된 책의 교환소 버튼 노출, 교환 가능 그린닷)
- 최종보상 삽화 스티커 (챌린지 6단계 클리어 시 명예 스티커, 터치 시 정보 표시)

#### C. 스토리 던전 단계 선택 화면
- 타이틀 영역 (챕터명)
- 보유재화 및 메뉴바 영역 (코인 획득 가능 이모지, 보유량 변경, 하트/메뉴바 동일)
- 버튼 영역 (합류 이벤트, 신규 쿠키 획득 가능 표시/이동, 픽업쿠키 이벤트 이동, 교환소, 스토리 다시보기)
- 에피소드 목록 (Ep1~6, 잠금/오픈 상태, 잠금 해제 조건, 진행도 표시)
- 게임정보 표시 영역 (Ep1-2: 전투력, Ep3-6: 전투력/추천 쿠키/예상 소탕 보스/쿠키 조합/PLAY 버튼/지난 시즌 Ep3-6 소탕 가능)
- 추천 장비 팝업
- 다시보기 팝업 및 스토리

#### D. 챌린지 모드 단계 선택 화면
- 타이틀 영역 (챌린지모드 표기)
- 단계 목록 (스토리모드와 동일한 규칙, 스크롤, 단계 선택 시 강조, 잠금 처리)
- 현재 단계 표시
- 게임 정보 표시 영역 (전투력/보상/추천 쿠키)
- PLAY 버튼 (4-6단계 쿠키 미보유시 비활성 토스트, 이전 단계 미클리어 시 비활성 토스트)

#### E. 상점 (코인 교환소)
- 타이틀 영역 (교환소 제목 - 스토리코인 교환소)
- 교환소 변경까지 남은시간 (dd:hh)
- 재화바 (현재 챕터 스토리코인 표시)
- 상품 영역 (이름/획득개수/가격/재고량, 재화 부족 시 딤드 및 팝업, 매진 SOLD OUT)
- 구매팝업 및 상점 NPC

#### F. 픽업 쿠키 상품 팝업 및 아이콘 노출
- 노출 상품 (챕터의 주인공 쿠키 관련 상품 - 레전 쿠키/에픽 쿠키 출석 팩)
- 팝업 및 아이콘 노출 조건 (Ep.2 클리어 후 에피소드 화면으로 돌아왔을 때 강제 팝업/아이콘)
- 상품 구매 후 종료 처리

### 중점 확인 사항

1. **컨텐츠 흐름 순서 준수**: 메인로비 → 모드리스트 → 최근책장 → 스토리던전 단계선택 → 챌린지모드 단계선택 → 상점 순으로 작성
2. **각 영역의 강조 번호 순서대로 작성**: 이미지의 녹색 사각형/동그라미 + 숫자 매칭 순서대로 sub2/sub3 구성
3. **화면 전환 케이스 누락 방지**: PLAY, 교환소, 합류 이벤트, 픽업 이벤트, 다시보기 등 각 버튼의 터치 → 화면 전환/팝업 케이스 빠짐없이 작성
