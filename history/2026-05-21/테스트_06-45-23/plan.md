### 기능 목록

**[A] 수면(Sleep) 시스템**
1. 수면 걸림 조건 (달빛술사 쿠키 공격 / 오르골 작동)
2. 수면 적용 대상 (중간보스/보스 제외 모든 몬스터)
3. 수면 적용 제외 대상 (쿠키, 중간보스, 보스)
4. 수면 상태 처리 (스턴과 동일, BT 처음으로 복귀, 스킬 사용 중단)
5. 수면 전용 애니메이션 재생 (Parameter: Sleep)
6. Fly 몬스터 추락 처리
7. 수면 해제 조건 1 - 시간 경과(N초)
8. 수면 해제 조건 2 - 피격(틱뎀 제외)
9. 틱뎀(DoT)으로 인한 데미지는 수면 유지
10. 수면 해제 후 디버프 부여 (달빛술사 공격 피해 증가)
11. 수면 이펙트 출력 (몬스터 머리 위, 동시 다수 적용 가능)
12. 전용 수면 애니메이션 몬스터 (꿈꾸는 모빌, 노래하는 인형, 꿈을 쏟아내는 모빌)

**[B] 수면 보호막 시스템**
1. 보호막 사용 몬스터 (노래하는 인형)
2. 보호막 부여량 산정 (부여자 최대 체력 기준)
3. 보호막 해제 조건 1 - 수면 적중 시 즉시 사라짐
4. 보호막 해제 조건 2 - 일반 피격으로 게이지 소진
5. 보호막 게이지 바 UI 출력 (이펙트에 어울리는 색상)

**[C] 오르골(Musicbox) 기믹**
1. 오르골 배치 (3m × 3m 크기, 스페셜 챕터)
2. 상태 흐름 (비활성화 → 활성화 → 작동 → 비활성화 순환)
3. 활성화 상태 - 실선 범위 표시 (반지름 10.5m)
4. 활성화 상태 - 게이지 충전 (6초)
5. 게이지 충전 영역 진입 시 차오름 시작
6. 영역 내 적 사망 시 게이지 0.5초 단축
7. 영역 이탈 시 게이지 감소 (Delay 설정 가능)
8. 게이지 충전/감소 색상 분기 (차오름: 푸른/노란 빛, 감소: 붉은 빛)
9. 게이지 가득 참 → 작동 상태 전환
10. 작동 효과 - 화면 전체 Enemy 수면 디버프 부여
11. 작동 애니메이션 (약 0.8초, 별 회전+빛 발산)
12. 수면 퍼짐 이펙트 (Active_Aura, 최대 반지름 8m, 1초~)
13. 작동 종료 후 비활성화 전환
14. 비활성화 상태 - 실선/게이지 미표시
15. 비활성화 시간 경과 후 활성화 재전환 (N초, GmTile 별 설정)
16. 활성화 애니메이션 (별/태엽 회전, 음악 재생)
17. 비활성화 애니메이션 (별 멈춤, 회색 변화)
18. 활성화 이펙트 (별 자체 발광)
19. 바닥 선/바닥 차오름 이펙트

**Priority 분배 근거**
- Critical: 핵심 진입/조작 경로(오르골 작동 전환, 화면 전체 수면 부여, 보호막 해제), 게임 진행 핵심 기능
- Major: 동적/조건부 처리(게이지 충전·감소, 수면 해제 조건, 디버프 부여, 보호막 산정 등)
- Normal: 단순 정적 출력(애니메이션, 이펙트, 색상 표현 등 연출 요소)

### 중점 확인 사항

1. **수치/조건 정확성**: 수면 지속 시간(5초), 활성화 반지름(10.5m), 게이지 충전 시간(6초), 적 사망 시 단축(0.5초), 수면 퍼짐 반경(8m), 작동 애니(0.8초) 등 기획서 명시 수치 반영
2. **상태 전이 흐름**: 오르골의 비활성화 → 활성화 → 작동 → 비활성화 순환 구조의 각 전환 시점 검증 TC 작성
3. **제외 대상 명확화**: 중간보스/보스/쿠키는 수면 적용 제외, 일반 몬스터만 적용되는 점을 별도 TC로 분리
4. **틱뎀과 일반 피격 구분**: DoT 데미지는 수면 유지, 그 외 데미지는 수면 해제되는 분기 처리
5. **Fly 몬스터 특수 처리**: 수면 적용 시 추락 동작 별도 TC
6. **수면 해제 후 디버프**: 달빛술사 한정 피해 증가 디버프 (다른 쿠키 피해는 증가하지 않음)
7. **전용 애니 몬스터 3종**: 꿈꾸는 모빌, 노래하는 인형, 꿈을 쏟아내는 모빌
8. **보호막 해제 이중 조건**: 수면 적중 시 즉시 제거 vs 일반 피격 시 게이지 소진
9. **게이지 시각 분기**: 차오름(푸른/노란 빛) vs 감소(붉은 빛) 색상 변화
10. **작동 효과 범위 차이**: 수면 적용 범위(화면 전체) vs 이펙트 표시 범위(반지름 8m)는 다름
11. **GmTile 별 비활성화 시간 설정**: 동일 오르골이라도 타일별로 다른 시간 적용 가능
12. **인게임 제외 규칙 준수**: 인게임 내부 전투 로직이지만 기믹 시스템 자체의 동작이므로 시스템 기능 위주로 작성하며, 단순 인게임 플레이 진행은 제외
13. **이미지에서만 확인된 UI 요소**: 보호막 게이지 바의 빨강+보라 빗금 패턴, 인게임 디버그 패널은 빌드 명세 부족으로 TC화 시 일반화 표기
