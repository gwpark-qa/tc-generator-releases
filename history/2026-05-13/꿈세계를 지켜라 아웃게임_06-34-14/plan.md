### 기능 목록

**1. 튜토리얼**
- 이벤트 모드 첫 진입 시 스토리 다이얼로그 1회 재생
- 유실물 상자 보유 후 이벤트 모드 진입 시 상자 선택 튜토리얼 발동

**2. 이벤트 진입/종료 (시간 관련)**
- starttime 도래 시 모드리스트 노출
- endtime 도래 시 모드리스트 제거
- 종료 시점 대기/매칭중 유저 기차 로비로 강제 이동 + 토스트 메시지
- 종료 30분 전 매칭 lock
- 종료 시 미수령 보상 메일 자동 지급
- 종료 1일 이하 시 타임바 빨간색 표기

**3. 기차 로비 진입점**
- 이벤트 버튼 진입
- 이벤트 시작 시 느낌표닷 / 받을 보상 있을 때 레드닷
- 플레이 버튼 상자 보상 노티 (오픈 가능 / 배치 가능)

**4. 이벤트 플레이 로비 (메인 허브)**
- 이벤트 타이틀, 시즌 정보, 종료 남은 시간 (2단위) 표시
- 시즌별 서브 타이틀 변경
- 모드리스트 첫 노출 시 NEW 배지
- 인포(i) 버튼 → 가이드 팝업
- 유저 정보 영역 (쿠키, 칭호, 닉네임, 길드명)
- 유저 상태 표시 (준비 중 / READY / 쿠키 설정 중)
- 보유 재화 표시

**5. 가이드북**
- 가이드북 진입
- 탭 구성 (이벤트 모드 / 라운드 규칙 / 보상 상자)
- 난이도별 설명, 상자 획득/확률, 단축권 사용법

**6. 랭킹**
- 리더보드 탭 (상위 1000위 노출, 상위 10위 노란색, 본인 초록색)
- 100위 이하 시 "???" 표시
- 무기록자 참여 독려 텍스트
- 랭크 보상 탭 (등급별 아이콘 C/B/A/S/S+/L/L+, 수령 / 딤 처리)

**7. 도감**
- 쿠키 도감 탭 (해금/잠금/선택/미션 달성 상태)
- 드래프트 도감 탭 (사용/미사용/히든)
- 도감 달성도 진행/수령 가능/수령 완료
- 신규 등록 시 도감 토스트
- 자세히 보기 / 해금 조건 표시
- 시즌 종료 시 도감 초기화

**8. 주간 미션 (주간 스탬프)**
- 미션 진행도 표시
- 스탬프 UI, 레드닷 표시
- 보상 수령
- 수요일 9:00 주간 초기화

**9. 유실물 센터 / 슬롯**
- 슬롯 상태 (수령 가능 / 진행 중 / 빈 슬롯)
- 상자 선택 팝업 (리스트, 선택, 보유 0개 시 미보유 표기, 선택하기 버튼)
- 상자 확률/대체 보상 확률 상세 팝업
- 진행 중 상자 시간 표기
- 정화 중단 불가 (한 번 장착 시 해제 불가)
- 보상 획득 연출 (페이드 아웃 → 보상 노출)
- 시간 단축권 사용 팝업 (토글바, MAX, 필요 수량 초과 차단)
- 유실물 리스트 팝업 (마일스톤, 보상 리스트, 대체 보상, 보너스 보상)
- 마일스톤 진행도 / 다음 마일스톤 보상 표기

**10. 쿠키 선택**
- 쿠키 리스트 출력 (성급/레벨/타입)
- 선택된 쿠키 표시 (딤드)
- 자세히 보기 화면
- 무작위 픽 (최상단 정렬, 인포 툴팁, 다른 유저 미선택 쿠키 중 랜덤)
- 미보유 쿠키 매우 낮은 확률 분기
- 모든 유저 선택 완료 후 공개

**11. 매칭 / 플레이 버튼**
- PLAY 버튼 (매칭 시작)
- 파티 전원 PLAY 시 매칭 진행
- 매칭 중 상태 (mm:ss 진행 시간)
- 매칭 취소(X) 버튼
- 매칭 완료 → 3초 후 인게임
- 3명 파티 시 매칭 스킵
- 쿠키 중복 허용 매칭 시간별 차등 (30/60/90/130초)
- 챔레일 영역 (보유 상자/단축권 개수)

**12. 채팅**
- 채팅 버튼 / 파티 채팅

**13. 인게임 로딩창 (아웃게임 영역)**
- 유저 카드 정보 표시
- 무작위 픽 쿠키 노출
- 로딩 문구 / 로딩바

**14. 인게임 결과창 (RESULT)**
- 타이틀, 라운드, 플레이 타임
- 점수 / New Record / Grade 표시
- 본인 정보 최상단 정렬
- 쿠키 영역 (본인 중앙, 닉네임 색상 본인 초록/타인 흰색, Disconnected 표기)
- 드래프트 표시 (메인 최대 3개, 서브)
- 보상 표시 (BG/아이콘/카운트, 툴팁)
- 다시하기 버튼 (카운트다운, 비활성화, 전원/일부 다시하기 분기)
- 나가기 버튼
- 이모지/신고 버튼
- 함께한 유저 다시 초대 팝업 (친구 초대, 파티 초대, 하루 보지 않기)

**15. 예외 처리**
- 개별 발주/디스커넥션 (재접속 복귀, 보상X/기록X)
- 전원 발주 (강제 종료)
- 중단하기 투표 (3명/2명 파티 2명 이상 동의, 1명 단독 동의)

### 중점 확인 사항

1. **카테고리 통합 — 단일 main으로 묶기**
   - "이벤트 플레이 로비"를 main 카테고리로 두고, 가이드북/랭킹/도감/주간 미션/유실물 센터/쿠키 선택/플레이 버튼을 모두 sub1으로 포함. 별도 main으로 분리하지 않음.
   - 단, "기차 로비"(이벤트 진입점)와 "인게임 로딩창", "인게임 결과창", "예외 처리"는 흐름·맥락상 별도 main으로 분리.

2. **이벤트 시간 관련 예외 — 분기별 처리**
   - 종료 시점에 위치(대기/매칭/인게임)별로 처리 결과가 다르므로 각 상태별 TC를 분리 작성.
   - 매칭 lock 시점(종료 30분 전), 미수령 보상 메일 지급은 Critical/Major로 분류.

3. **매칭 분기 처리 — 인원/시간/상태별**
   - 1인/2인/3인 파티별 동작 차이, 매칭 진행/취소/완료의 상태 전이, 쿠키 중복 매칭 시간별(30/60/90/130초) 차등 룰을 빠짐없이 포함.

4. **상자/유실물 시스템 — 상태와 분기가 가장 복잡**
   - 슬롯 3상태(수령 가능/진행 중/빈 슬롯)별 UI 표기
   - 상자 보유 0개 시 미보유 표기·선택 차단
   - 정화 시작 후 중단 불가 처리
   - 시간 단축권 수량 초과 선택 차단, MAX 버튼 동작
   - 모든 유실물 수령 후 대체 보상 리스트 전환

5. **도감 상태 분기 — 쿠키/드래프트 패널별 4상태**
   - 잠금 해제/잠금/선택/미션 달성 4가지 시각 상태가 모두 노출되도록 TC 분리.
   - 신규 등록 시 도감 토스트, 느낌표닷 표기·제거 조건도 포함.

6. **결과창 분기 — 다시하기/나가기/Disconnected**
   - 전원 다시하기 vs 일부 다시하기 vs 일부 나감의 매칭 로직이 다르므로 각각 분리.
   - 본인/타 유저 닉네임 색상, 본인 상단 정렬, 디스커넥트 표기 케이스 명확히 작성.

7. **아웃게임 범위 준수**
   - 인게임 조작·드래프트 선택 동작·꿈석탄 운반 등 인게임 메커닉은 제외.
   - "PLAY 버튼 → 로딩 진입"까지만 작성하고 그 이후 인게임 진행 로직은 작성하지 않음.
   - 단, 결과창은 아웃게임 UI로 간주하여 작성.

8. **비기능 케이스 제외**
   - 해상도, 노치, 폰트 등 UI 외관 케이스는 작성하지 않음.

9. **Priority 보정 원칙**
   - Critical은 "결제 / 데이터 손실 / 진입 차단 / 게임 진행 불가" 직결 케이스에 한정.
     - 예: 이벤트 종료 시 미수령 보상 메일 미지급, PLAY 버튼 매칭 미시작, 보상 수령 시 미지급.
   - 단순 UI 출력/표기 케이스는 Normal로 배치.
