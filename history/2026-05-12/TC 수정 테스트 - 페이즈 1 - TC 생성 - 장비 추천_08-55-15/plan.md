### 기능 목록

**1. 쿠키 상세화면 진입점**
- 쿠키 상세화면 하단에 '추천 조합' 버튼 노출 (보유/미보유 쿠키 동일)
- 버튼 터치 시 추천 조합 팝업 진입
- 진입한 쿠키에 자체 포커싱

**2. 추천 조합 팝업 공통 UI**
- 상단 헤더 (타이틀 / 닫기 X 버튼)
- 좌측 탭 메뉴 (장비 / 아티팩트 / 잠재력 — 3개 탭 전환)
- 추천 쿠키 사이드바 (두 줄 + 스크롤, 사용중/미보유 구분, 등급별 컬러)
- 필터 버튼 (콘텐츠별 필터링 팝업 진입) 및 현재 필터 적용 상태 표시
- 새로고침 버튼 (페이지 정보 갱신)
- 해당 콘텐츠 바로가기 버튼 (챔피언스 레이드 등)

**3. 추천 장비 탭 (메인 대상)**
- 상위 3개의 추천 장비 조합 카드 출력 (1~3순위)
- 각 카드 표시 정보: 순위 / 사용률(소수점 첫째자리) / 무기·모자·상의·하의 4종 아이콘
- 등급별 외곽 컬러 적용
- 데이터 기준: 챔피언스 레이드 상위 1000명 사용률 기반
- 24시간 단위 데이터 갱신
- 카드 일러스트 터치 시 장비 상세 정보 진입
- 이미 장착중인 장비 일러스트 터치 시 "이미 사용중인 장비입니다." 토스트

**4. 조합 따라하기 기능**
- 케이스1: 조합 장비 모두 보유 + 다른 쿠키 미점유 → 전투력 최고 장비 자동 장착, "조합이 적용되었습니다." 토스트
- 케이스2: 일부 미보유 또는 일부 다른 쿠키 점유 → 가능 부위만 장착, 나머지는 기존 유지, "조합이 적용되었습니다." 토스트
- 케이스3: 장착 가능 장비 0개 → 장착 X, "장착할 수 있는 장비가 없습니다." 토스트
- 미보유 쿠키 → 조합 따라하기 버튼 비활성화

**5. 제작소 바로가기**
- 제작소 진입 버튼 터치 시 해당 장비 제작소 이동
- 제작소 미해금 시 안내 토스트 출력

**6. 데이터 예외/에러 처리**
- 누적 플레이 기록 1,000명 미만 → "기록이 충분하지 않습니다." 안내
- 상위 1000명 중 특정 부위 장비 종 3개 이내 → 사용 장비 누락 표시
- 데이터 로드 실패 → "데이터를 불러오지 못했습니다." 안내

### 중점 확인 사항

1. **진입점 검증 (Critical 핵심)**
   - 보유/미보유 쿠키 모두에서 '추천 조합' 버튼이 노출되어야 함
   - 미보유 쿠키에서도 팝업은 진입되며, 단 '조합 따라하기' 버튼은 비활성화 처리되는 조건 분기 명확화

2. **조합 따라하기 3가지 분기 (Critical 집중 영역)**
   - 분기1: 모두 보유 + 미점유 → 전투력 최고 장비 자동 장착 + "조합이 적용되었습니다." 토스트
   - 분기2: 일부 보유 / 일부 다른 쿠키 점유 → 가능 부위만 장착, 나머지 기존 유지 + 토스트
   - 분기3: 모두 불가 → 장착 X + "장착할 수 있는 장비가 없습니다." 토스트
   - 각 분기별 토스트 메시지 문구 정확성도 별도 검증

3. **데이터 표시 정확성 (Major)**
   - 사용률 소수점 첫째자리 반올림 표기
   - 상위 3개 조합만 표시 (4위 이하 노출 금지)
   - 무기/모자/상의/하의 4종 부위 모두 노출
   - 등급별 외곽 컬러 적용

4. **데이터 예외 케이스**
   - 챔피언스 레이드 누적 플레이 1,000명 미만 → "기록이 충분하지 않습니다." 안내
   - 상위 1000명 중 특정 부위 장비 종 3개 이내인 경우 누락 표시 처리
   - 데이터 로드 실패 → "데이터를 불러오지 못했습니다." 안내

5. **자체 포커싱 동작**
   - 쿠키 상세화면에서 진입한 쿠키가 사이드바에서 자동 선택(포커싱) 상태로 표시

6. **단일 기대결과 원칙 준수**
   - 탭 전환, 사이드바 쿠키 선택, 필터/새로고침 등 복합 동작은 결과별로 TC 분리

7. **이미 사용중 장비 토스트**
   - 추천 카드 일러스트 터치 시 본인 쿠키에 이미 장착된 장비라면 "이미 사용중인 장비입니다." 토스트

8. **제작소 진입 분기**
   - 제작소 해금 상태 → 해당 장비 제작소 이동
   - 제작소 미해금 상태 → 안내 토스트 출력

9. **데이터 갱신 주기**
   - 24시간 단위 자동 갱신 동작 및 안내 문구 표기

10. **TC 작성 범위 제한**
    - 타이틀이 "장비 추천"이므로 아티팩트/잠재력 탭은 **탭 전환 동작**만 검증하고, 각 탭 내부의 세부 기능 TC는 본 문서에서 제외 (향후 별도 페이즈에서 작성)
