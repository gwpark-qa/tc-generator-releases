이미지를 분석했습니다. 기획서와 UI 플로우를 교차 검증하여 TC를 작성하겠습니다.

### 기능 목록
1. 도전 미션 버튼 — 어비스 레이드 로비에 신규 배치, 다른 화면에는 표시되지 않음
2. 도전 미션 팝업 — 현재 시즌 미션 리스트 표시, 미션 조건 텍스트/횟수, 획득 보상(아이콘/개수), 받기 버튼(딤/활성화), 정렬 규칙
3. 보상 수령 — 조건 달성 시 즉시 유효 재화 지급, 어비스 퀘스트/마일스톤과 공유하지 않음

### 카테고리 구조
```
├─ 도전 미션
│  ├─ 도전 미션 버튼
│  │  ├─ 버튼 표시
│  │  └─ 팝업 진입
│  └─ 도전 미션 팝업
│     ├─ 미션 리스트
│     │  ├─ 미션 조건 텍스트
│     │  ├─ 미션 조건 횟수
│     │  └─ 정렬
│     ├─ 획득 보상
│     │  ├─ 보상 아이콘
│     │  └─ 보상 개수
│     ├─ 받기 버튼
│     │  ├─ 딤 처리
│     │  ├─ 활성화
│     │  └─ 보상 수령
│     ├─ 닫기 버튼
│     └─ 어비스 퀘스트 독립성
```

### TC 건수 배분
- 도전 미션 버튼: 약 5건
- 도전 미션 팝업 — 미션 리스트: 약 8건
- 도전 미션 팝업 — 획득 보상: 약 4건
- 도전 미션 팝업 — 받기 버튼: 약 8건
- 도전 미션 팝업 — 닫기/독립성/기타: 약 5건
- 합계: 약 30건

### 중점 확인 사항
- 받기 버튼의 딤/활성화 상태 전환 및 보상 수령 정합성
- 보상 획득 가능 미션 최상단 / 완료 미션 최하단 정렬 규칙
- 어비스 퀘스트와 마일스톤을 공유하지 않는 독립성 검증
- 조건 달성 시 즉시 유효 재화 지급 여부
- 도전 미션 버튼이 어비스 레이드 로비에만 표시되는지 확인
