### 기능 목록

1. **이벤트 ON/OFF** — start_time_stamp / end_time_stamp 기반 이벤트 활성·비활성, 엔드타임 도달 시 이벤트 종료 및 로비 튕김
2. **이벤트 UNLOCK** — 공용 이벤트(CONTENTS_TYPE_EVENT) 오픈 시점에 함께 오픈
3. **이벤트 진입 경로** — (1) 로비 > 이벤트 배너, (2) 공용 이벤트 > 2주년 이벤트 탭 > 이벤트 버튼, (3) 모드리스트 > 2주년 이벤트 대문
4. **2주년 로비 구성** — 케이크 이벤트(방꾸미기), 2주년 이벤트 상점, 스토리 이벤트 바로가기 숏컷, 신규 플레이블 이벤트 바로가기 숏컷, 어드벤처 바로가기 숏컷
5. **방꾸미기 이벤트 화면 진입** — 로비 방꾸미기 이벤트 버튼 터치 시 이벤트 화면 이동
6. **멜랑크림 NPC** — 어린 멜랑크림 일러스트 포트레이트 배치, 조건별 대사 말풍선(N초 마다 랜덤 재생), 데코 미완성 시 대사, 데코 최종 완성 시 대사 및 포즈/표정 변경
7. **포인트 정보** — 내 누적 포인트 표시, 유저 전체 누적 포인트 표시, 인포 버튼 터치 시 포인트 획득 관련 설명 팝업
8. **다음 가구 정보** — 다음 가구 배치에 필요한 누적 포인트 표시, 다음 배치 가구 리소스 이미지, 모든 가구 완료 시 "모든 가구 배치 완료" 텍스트 표시, 보상 아이콘 상태(도전중/획득 가능 강조/획득 불가 기본)
9. **가구 배치 보상 팝업** — 마일스톤 보상 표시, 현재 누적 포인트, 보상 아이콘/수량, 획득 가능 보상 강조 표시 및 터치 시 일괄 획득, 가구 배치 수 및 필요 누적 포인트 표시
10. **포인트 획득 미션** — 누적 포인트 기반 보상 획득, 일일 미션(매일 갱신), 미션 목록 및 진행도 표시
11. **가구 꾸미기 미션** — 가구 선택/가구 이미지·이름 표시, 레벨별 누적 포인트 게이지, 가구 꾸미기 가능 여부 확인, "다른 가구 보기" 버튼, 가구 배치 실행
12. **모든 가구 배치 완료 연출** — 완성 연출 재생, NPC "참 잘 완성이야~" 대사, "모든 가구 배치 완료" 텍스트, 스토리 바로가기 버튼 노출 → 2주년 스토리 로비 이동
13. **전체 유저 누적 포인트 연동** — 전체 유저 합산 포인트에 따라 가구 배치 및 보상 변동

### 중점 확인 사항

1. **이벤트 종료 시 강제 이동 처리** — 이벤트 화면(방꾸미기, 가구 배치 보상 팝업, 가구 꾸미기 미션 등) 각 서브 화면에서 엔드타임 도달 시 로비로 정상 튕기는지 모든 화면별로 검증 필요

2. **전체 유저 누적 포인트 ↔ 개인 포인트 구분** — 가구 배치는 전체 유저 합산 포인트 기반, 보상 마일스톤은 개인 배치 수 기반으로 서로 다른 기준이 적용되므로 혼동 없이 각각 정확히 반영되는지 확인

3. **보상 아이콘 3가지 상태 전환** — 도전중(기본) → 획득 가능(강조 색상) → 획득 완료 상태 전환이 조건에 맞게 정확히 이루어지는지 케이스별 검증

4. **가구 배치 보상 일괄 획득** — 다수의 보상이 동시에 획득 가능한 상태일 때, 터치 1회로 모든 획득 가능 보상이 누락 없이 지급되는지 확인

5. **멜랑크림 NPC 대사 분기** — 데코 미완성 상태 대사 랜덤 재생과 데코 최종 완성 시 대사/포즈 변경이 정확한 조건에서 전환되는지 확인 (특히 모든 데코 미션 완성 판정 기준)

6. **일일 미션 갱신 시점** — 매일 갱신 시 기존 미완료 미션 초기화 여부, 갱신 시점(서버 기준 00시 등)에 미션 화면을 보고 있는 경우의 처리

7. **가구 꾸미기 미션 → 방 반영 연출** — 가구 배치 실행 후 방꾸미기 이벤트 화면에 해당 가구가 실제로 반영되는 시점과 연출(애니메이션/효과음) 확인

8. **모든 가구 배치 완료 후 재진입** — 완성 연출은 최초 1회만 재생되는지, 이후 재진입 시 완성 상태의 방 + 스토리 바로가기가 유지되는지 확인

9. **3가지 진입 경로 동일 동작** — 로비 배너, 공용 이벤트 탭, 모드리스트 대문 어느 경로로 진입해도 동일한 2주년 로비로 이동하는지 검증

10. **케이크 이벤트 뼈대 활용에 따른 차이점** — 1주년 케이크 이벤트 구조를 사용하되 방꾸미기로 연출이 변경된 부분(가구 오브젝트 배치, 포인트 정보 UI 분리 등)이 정상 적용되었는지 확인
