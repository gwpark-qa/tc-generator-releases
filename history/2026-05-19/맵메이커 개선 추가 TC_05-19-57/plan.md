### 기능 목록

**대분류 1. 맵메이커 개선 - 세부 규칙 설정**
- 맵 만들기 > 나의 맵 > 편집하기 진입 화면 (맵 타이틀, 환경/닫기 아이콘, 공개하기/편집하기 버튼)
- 룰(니바퀴) 버튼 터치 시 맵 설정 팝업 출력
- 맵 설정 팝업 좌측 탭: 맵 정보 / 플레이어 설정 / 추가 규칙
- 추가 규칙 탭 - 최대 3개까지 적용 가능 (적용된 규칙 3/3 표시)
- 규칙 리스트(아이콘+규칙명+설명+수치+적용 버튼) 출력
- 규칙 3개 적용 시 4번째 [적용] 버튼 딤드
- [기본값으로] / [저장하기] 버튼 동작
- [저장하기] 미클릭 시 규칙 미반영
- 맵 상세보기 화면 우상단에 적용 규칙 아이콘 표시 (1개/2개/3개 분기)
- 규칙 아이콘 터치 시 "추가규칙 보기" 팝업 출력 (적용된 규칙 항목별 노출)
- 플레이어 설정 탭 - 시간 수정 (숫자 < > 조절, 이하 < > 조절, 적용하기)
- 숫자 터치 시 말풍선(직접 입력/MAX 버튼) 출력

**대분류 2. 클리어 조건 설정**
- 맵 설정 팝업 진입 시 첫 화면 기본 설정 탭 위치
- 플레이어 설정 탭 - 클리어 조건 영역
- 젤리 획득 미션 (아이콘, 숫자, 이상/이하/일치 선택)
- 클리어 타임 미션 (숫자, 이상/이하 선택)
- 클리어 조건은 1개만 적용 가능
- [적용하기] 터치 시 [✓ 적용중] 상태 전환
- [추가] / [저장하기] 버튼 동작
- 젤리 아이콘 터치 시 말풍선(젤리 종류 스크롤 변경) 출력
- 인게임 시작 시 클리어 조건 미션 팝업 출력 후 사라짐
- 인게임 상단 좌측에 보너스 타임 게이지 + 카운트(0/50) 출력
- 젤리 미션 카운트 표시 (N개 이상/이하/일치 분기별 ✅/❌)
- 시간 미션 카운트 표시 (N초 이상/이하 분기별 ✅/❌)
- 결과화면 PERFECT 표기 / 클리어 조건 일부 실패 X 표기

**대분류 3. 맵 찾아보기 개선**
- 맵 찾아보기 메인 진입 (상단 통화, 설정/홈/메뉴)
- 상단 탭 (신규맵 / 명예의 전당 / 엘도라 찾기 / 나의 맵 찾기 / 내맵고)
- 맵 카드(좋아요/플레이/작성자 ID/맵 이름) 출력
- 나의 맵 표시 인포 버튼 터치 시 안내 팝업 출력
- 맵 그리드 스크롤
- PICK 진입 축하 팝업 출력
- 맵 카드 터치 시 달려보기 진입
- 내 맵 홍보하기 팝업 (홍보 가능 슬롯, 안내문, 맵 리스트)
- 홍보하기 버튼 터치 시 💎2,000 소비 + 24시간 카운트다운
- 홍보중 상태 버튼 변경 (홍보 후 23:59:59 남음)
- 명예의 전당 등록 안내 / 랭킹 Top 3 강조

**대분류 4. 랭킹 추가 기능**
- 랭킹 메인 화면 - 맵 만들기 버튼
- 맵 점수 랭킹 / 퍼펙트 클리어 표시
- 내 기록 없을 시 안내 문구
- 내 랭킹 Top 3 진입 시 강조 표시
- 명예의 전당 점수 1등 맵 개수 랭킹

**대분류 5. 맵 심사하기 개선**
- 심사하기 진입 시 상단 배너 (오늘의 심사 횟수 0/3)
- 맵 후보 3개 가로 배열 (작성자/등록일/맵이름)
- 진입 시 맵 등장 연출 (우→좌)
- 타이틀 펄럭 연출
- 셀 선택 시 1.2배 확대 유지
- [확정하기] 노란 버튼 노출
- 확정 시 % 표시 (선택률)
- 미선택 맵 사라지고 선택 맵 중앙 이동, 캐릭터/말풍선 등장
- [심사하기] 버튼 출력
- 심사 횟수 소진 시 안내 + 99:99:99 카운트다운
- 일시정지 팝업 - 계속하기/그만하기 버튼
- 평가 패널 - 별로에요/좋아요 + 즐겨찾기 등록/신고하기
- 평가 불가 상태 - 자물쇠 표시 + "아직 평가할 수 없어요"
- 선택 시 좋아요/싫어요 강조
- 즐겨찾기 등록 시 ★ 강조 / 취소 시 해제
- 결과 화면 - Result 타이틀, 점수/젤리/심사 보상 ♥50
- 결과 화면 - 평가 옵션

**대분류 6. 인기맵/명예의 전당 탭 (10번)**
- 상단 탭 - 추천맵/인기맵/신규맵/명예의 전당/코드로 찾기/즐겨찾기
- 인기맵 탭 - 최근 7일 평가받은 100개 맵 노출
- 7일간 평가받지 않은 경우 "-" 표시
- 인기맵 결과발표 팝업 (7일 경과 후 진입 시) - 인기 점수 / 명예의 전당 후보 / 상위 30% / 보상
- 신규맵 탭 - NEW 뱃지 / 새로고침 버튼
- 신규맵 상세 확인 시 NEW 사라짐
- 명예의 전당 탭 - 최신달 / 후보달 화살표 전환

**대분류 7. 맵 공개 성공 & 즐겨찾기 (11번)**
- 맵 공개 성공 팝업 - 평가 결과 표시 / 인기도 도전 안내
- 황금셀 터치 시 팝업 출력
- 맵 상세 - 즐겨찾기 등록/취소
- 즐겨찾기 최대 개수 처리

**대분류 8. 치트 퍼펙트 랭킹 & 토스트 (12번)**
- 치트 퍼펙트 성공 횟수 랭킹 - 전체/월간
- 빈 벨 슬롯 터치 시 토스트 + 즐겨찾기 추가
- 채워진 벨 슬롯 터치 시 토스트 + 즐겨찾기 제거

**대분류 9. 조합 고정 & 능력 ON/OFF (13~15번)**
- 맵 설정 - 조합 및 능력 탭
- 쿠키/펫/보물 능력 사용 토글 (ON/OFF)
- 고정 조합 사용 - [고정조합 설정] 버튼
- 초기 상태: 능력 ON, 고정조합 없음
- 고정 조합 설정 - 캐릭터 그리드 선택 / [조합 저장하기] / [해제하기]
- 조합 저장 후 설정 화면에 캐릭터 아이콘 + 🗑 삭제 버튼 표시
- [해제하기] 터치 시 "초기화할까요?" 안내 + 리셋
- 쿠키 조합 변경 화면 (검색/정렬 드롭다운)
- 고정된 조합은 변경 불가
- 반복펫은 각각 다른 개체

**대분류 10. Result 화면 & 맵 상세정보 (16~17번)**
- Result - 클리어 조건 체크리스트 (✅/❌)
- 슬롯이 하나씩 뜨고 체크/X 표시
- PERFECT! 도장 연출
- PERFECT 옆 ▼/▲ 토글 - 평가/달성목록 펼침
- 평가 옵션 - 별로에요/좋아요
- 맵 상세정보 - 맵 코드/공개하기/편집하기
- 규칙 상세 정보 팝업

**대분류 11. 명예의 전당 발표 & 코드로 찾기 (18~19번)**
- 명예의 전당 발표 팝업 (KST 0시) - 상위 3개 맵 카드
- 후보 100개 맵 표시 - 자동 스크롤
- 내 기록 없을 시 / 있을 시 분기
- 코드로 찾기 탭 - 입력 필드 / [찾기] 버튼
- 명예의 전당 등록 결과 - 상위 30% / 보상
- 일시정지 화면 (즐겨찾기 등록 포함)
- 즐겨찾기 탭 - 즐겨찾기 등록 맵 리스트

**대분류 12. 즐겨찾기 & 정렬 (20번)**
- 빈 별 슬롯 클릭 시 토스트 + 추가 안 됨
- 즐겨찾기 취소 가능 상태 - 일시정지 팝업
- 즐겨찾기 맵 최대 개수 도달 시 안내 + 추가 차단
- 정렬 드롭다운 - 별점순/이름순/좋아요순/플레이순

**대분류 13. 공개하기/달려보기/심사하기 변경 (21~22번)**
- 공개하기 화면 - 쿠키/펫/보물 능력 OFF
- 능력 ON 시 등록테스트 버튼 상단 문구 "ON!"
- 달려보기 화면 - 동일 레이아웃
- 심사하기 진입 - 캠프 배경 + 심사하기 버튼
- 심사하기 팝업 - 맵 이름/작성자/맵 코드/등록일/클리어 조건/PLAY 버튼
- 추가 규칙 1~3개 가변 표시
