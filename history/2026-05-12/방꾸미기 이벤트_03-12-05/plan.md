## 1단계: TC 설계

### 기능 목록

기획서에서 파악한 기능 (단, **2주년 이벤트 - 로비** 및 **2주년 이벤트 - 플레이블 이벤트**는 제외):

**이벤트 기본 시스템**
1. 이벤트 ON/OFF: start_time_stamp / end_time_stamp 기반 오픈/종료
2. 이벤트 종료 시 이벤트 화면 유저를 로비로 퇴장 처리
3. 이벤트 UNLOCK: 공용 이벤트(CONTENTS_TYPE_EVENT) 오픈 시점에 함께 오픈

**이벤트 진입 경로**
4. 로비 > 이벤트 배너로 진입
5. 공용 이벤트 > 2주년 이벤트 탭 > 이벤트 버튼으로 진입
6. 모드리스트 > 2주년 이벤트 대문으로 진입

**케이크 이벤트 (방꾸미기 핵심)**
7. 전체 유저 플레이 진행도에 따른 케이크 단(tier) 추가
8. 개별 유저 미션 달성에 따른 데코 추가
9. 케이크 탑 만들기 기능 (UI상 "케이크 탑 만들기" 버튼 확인)

**2주년 이벤트 상점**
10. 이벤트 재화로 보상 교환
11. 교환소 UI 진입 및 동작

**바로가기 숏컷**
12. 스토리 이벤트 바로가기 숏컷 (달술/밀키웨이/스타더스트 스토리 버튼 모음)
13. 어드벤처 바로가기 숏컷 (꿈열차 어드벤처 로비 이동)

> **참고**: 플레이블 이벤트 바로가기 숏컷(4번 항목)은 "플레이블 이벤트" 제외 범위에 해당하여 TC에서 제외합니다.

### 중점 확인 사항

1. **케이크 단 추가 동기화**: 전체 유저 진행도가 실시간/주기적으로 반영되는지, 서버-클라이언트 동기화 검증 필수 (Critical)
2. **개별 미션 달성 → 데코 추가**: 미션 달성 조건별 데코가 정확히 배치되는지, 데코 중복 지급/누락 여부 (Critical)
3. **이벤트 종료 시 퇴장 처리**: 엔드타임 도달 시 이벤트 화면 유저가 로비로 정상 이동하는지, 교환/만들기 중 종료 시 처리 (Critical)
4. **이벤트 상점 재화 교환**: 재화 부족/충분 시 교환 동작, 교환 한도, 재화 차감 정확성 (Critical)
5. **3가지 진입 경로**: 각 진입 경로별 동일한 이벤트 화면 도달 여부 확인
6. **2D 일러스트 배경**: 1주년(3D)과 달리 2주년은 2D 일러스트 배경 — 배경 리소스 정상 로드 및 해상도 확인
7. **숏컷 이동**: 각 숏컷 버튼 터치 시 올바른 목적지로 이동하는지, 이동 후 뒤로가기 시 이벤트 로비 복귀 여부
8. **레드닷 알림**: 수령 가능 보상, 미확인 신규 상품 등에 대한 레드닷 출력/제거 조건
