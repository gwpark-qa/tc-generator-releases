### 중점 확인 사항

1. **수면 시스템의 적용 대상 분기**
   - 일반 몬스터/엘리트(꿈꾸는 모빌)/중간보스/보스의 수면 적용 여부 명확히 구분
   - Fly 몬스터(꿈꾸는 모빌, 꿈을 쏟아내는 모빌)의 추락 처리
   - 틱뎀(DoT) 예외 처리: 연금술사/멜랑크림/드라이스톤 등 DoT는 수면 해제 트리거가 되지 않음

2. **오르골 게이지 상태 전이**
   - 비활성화 → 활성화 → 작동 → 비활성화 순환
   - 영역 진입/이탈/적 사망 시 게이지 변화 (색상 변화 포함)
   - 활성화 영역 반지름 10.5m, 수면 퍼짐 영역 8m (이펙트와 실제 수면 적용 범위 분리)

3. **수면 보호막의 소멸 조건 분기**
   - 수면 적중 시 즉시 소멸 vs 일반 피격 누적 시 소멸
   - 보호막 보유 중 수면 디버프 부여 시 보호막이 사라지는 동작
   - 보호막 중첩 불가 (이미 보유 시 타겟 제외)

4. **몬스터 패턴별 충돌/정지 조건**
   - 악몽의 아가양/악몽의 양 돌진: 타겟 지나침(90도 이탈), 낙사 지형, 벽 충돌 분기
   - 충돌 성공/실패 별도 애니 재생
   - 공격 실행 중 경직 면역 검증

5. **꿈꾸는 모빌/꿈을 쏟아내는 모빌의 수면 연계 효과**
   - 수면 시 소환 몬스터 전부 사망
   - 꿈을 쏟아내는 모빌의 5초간 지상 잔존 (추가 타격 가능 윈도우)
   - 평소 Hide 상태 vs 드림캐쳐 작동 시 타격 가능 전환

6. **보스 어비스레이드 핵심 검증**
   - 페이즈 전환 (1페이즈 낙사 불가 → 2페이즈 낙사 가능)
   - 허상 시스템: 모든 패턴에 허상 섞임, 인디케이터로 구분 불가
   - 심연의 손길: 속박 대상 없을 시 즉시 종료 (예외 처리)
   - 진짜 찾기: 분신 타격은 보스 스탯에 영향 없음
   - 악몽 폭주: 흡수한 악몽 개수에 비례한 데미지 정확 적용
   - 필살기 악몽 정화: 3단계 기믹(파괴/무력화/상쇄) 각 성공/실패 분기, 최종 정화 비율 판정

7. **거대한 악몽 램프의 페이즈 전환**
   - 저체력 시 카메라 후방 고정 + 정해진 패턴 공격으로 전환

8. **인디케이터 표시 정확성**
   - 각 스킬별 인디케이터 차오름 시간 (악몽 램프 2초, 거대 램프 Skill_A 2초/Skill_B 1.8초/Skill_C 1초 등)
   - 인디케이터 위치 정확성 (쿠키 위치 기준 십자 패턴)

9. **다중 배치 시 동작**
   - 악몽 램프 2마리 배치 시 1초 텀
   - 악몽의 아가양 3마리 이상 배치 시 다른 몬스터 수 조절
