### 중점 확인 사항

1. **5/13 스펙 변경 반영**: 능력 카운트 모두 소모 후 추가 공격 **불가**로 변경됨. 기존 "카운트 소모 후에도 반복 가능하되 젤리 미생성" 스펙이 폐기되었으므로, 카운트 소모 후 입력 차단이 정확히 동작하는지 반드시 검증 필요.

2. **왕만두 발사 미수행 시 마법사탕 능력 영향**: 개발 이력에서 "왕만두를 발사하지 않은 상태로 능력 종료 시 다음 마법사탕 능력에서 왕만두가 나가는 문제"가 수정됨. 왕만두 미발사→능력 종료→마법사탕 능력 진입 시나리오를 별도 TC로 검증.

3. **조작 연계 안정성**: 점프→공격→추가 점프→공격 연계, 착지 직전 공격 입력, 빠른 연타 입력 시 투사체 누락 등 타이밍 민감 케이스를 세밀하게 검증. 공중 공격 2회 제한과 추가 점프 1회 제한이 정확히 적용되는지 확인.

4. **마법사탕 카운트 소멸 조건**: 마법사탕 카운트를 보유한 채 본 능력에 진입하면 본 능력 종료 후 카운트가 모두 소멸되는지, 소멸 타이밍이 정확한지 검증.

5. **젤리 수치 정합성**: 
   - 본 능력: 소만두 15개, 왕만두 30개, 점수 1,001~1,015 (레벨당 +1)
   - 마법사탕: 점수 100~500 (레벨당 +100)
   - 펫: 작은 만두 1,001~1,015 / 왕만두 2,001~2,015 (레벨당 +1)
   - 각 레벨별 점수 증가폭이 정확히 반영되는지 전수 확인.

6. **버튼 활성화 시간 갱신**: 버튼 활성화 상태에서 추가 점프 시 활성화 시간이 갱신되는지, 비활성 상태에서 추가 점프 시 정상 동작하는지 분기별 확인.

7. **펫 능력 연기 이펙트 카메라 처리**: 젤리가 멈춘 후 연기 이펙트가 화면 오른쪽에 잔류하지 않는지 (affectCamera 처리), 투명도 설정값이 정상 적용되는지 확인.

8. **능력 발동 애니메이션→만두 들기 전환**: 능력 발동 애니메이션 종료 후 만두 들기 상태로 자연스럽게 연결되는지 (프레임 끊김 없이), 만두 들기 상태의 달리기 속도가 기본 달리기 속도와 일치하는지 확인.
