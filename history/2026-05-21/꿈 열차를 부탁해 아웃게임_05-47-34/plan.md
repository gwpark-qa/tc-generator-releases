### 기능 목록

**1. 모드리스트/진입**
- 이벤트 모드 진입 (꿈세계 배경, 멀어지는 기차)
- 모드리스트 알림 (느낌표닷/레드닷)
- 이벤트 종료 카운트다운

**2. 플레이 로비**
- 헤더 영역: 뒤로가기, 이벤트 타이틀, 인포(i), 이벤트 종료 카운트다운
- 상단 메뉴: 가이드북, 랭킹, 도감, 햄버거 버튼, 홈
- 탑바: 2주년 코인 재화바, 일반 재화, 메일/알림, 친구 등
- 메인 영역: 길드/유저 3인 슬롯 (캐릭터 모델링, 닉네임, 길드명)
- 유저 상태 표시 (기본/쿠키 세팅 중/READY)
- 채팅 아이콘
- 우측 패널: 내 최고 기록 (등급, 점수, 인포)
- 우측 패널: 피냐타(유실물) 정화 슬롯 (배치 가능/정화 중/완료)
- 우측 패널: 주간 유실물 미션 진행도/보상
- 하단: 주간 꿈세계 스탬프 진행도 + 보상
- 이벤트 상점 바로가기 버튼
- PLAY 버튼

**3. 매칭 시스템**
- 매칭 큐 등록 (매칭 중... 카운트)
- 매칭 취소 버튼
- 매칭 완료 상태
- 매칭 실패/타임아웃
- 매칭 중 네트워크 끊김

**4. 튜토리얼/안내**
- 차장 NPC PlayDialogue 대사 (최초 1회)
- 모드 라이트팝 4컷 가이드
- 가이드북

**5. 쿠키 선택**
- 쿠키 그리드 (10종, 5성 60레벨 고정)
- 정렬/필터 (추천순, 57/57)
- 쿠키 미리보기 (전신, Lv.60, 5성)
- 쿠키 정보 버튼 (돋보기)
- 쿠키 미션 영역 (왕관 아이콘)
- 무작위/히든 쿠키 슬롯 (?, 달빛술사 쿠키)
- 해금 쿠키 (미션 조건/기간 조건)
- 스타트 쿠키 4종
- 동일 쿠키 중복 선택 여부
- Shiny 이펙트 (미션 완료 시)

**6. 쿠키 자세히 보기 팝업**
- 쿠키 등급/속성/이름/역할/타입
- 스킬 패널 5종 (패시브/기본공격/특수스킬/대시/궁극기)
- 스킬 태그 표시
- 쿠키 승급 정보 버튼
- 팝업 닫기

**7. 인게임 진입**
- 로딩 화면

**8. 결과 화면 (RESULT)**
- 클리어 결과 표시
- 나가기 버튼 → 플레이 로비 복귀

**9. 함께한 유저 다시 초대하기 팝업**
- 유저 리스트 (3명, 닉네임, 점수/KDA)
- 친구 초대 버튼
- 파티 초대 버튼
- "오늘 하루동안 보지 않기" 체크박스
- 닫기 버튼

**10. 랭킹**
- 진입 (내 최고 기록 박스 터치, 인포 버튼)
- 랭킹 리스트 탭 (Rank/Player/Tier/Cookie/Score)
- 페이지네이션
- 랭킹 정보 탭 (등급별 보상 리스트)
- 기록 없음 케이스
- 랭킹 보상 레드닷

**11. 도감**
- 쿠키 도감
- 드래프트 도감
- 미션 보상 팝업

**12. 유실물 정화 슬롯**
- 슬롯 상태 5종 (NoItem/Empty/InProgress/Complete/SlotOpen)
- 정화 중 이펙트
- 정화 완료 이펙트
- 타임 게이지/잔여시간 표시
- 뉴 닷/레드 닷 알림
- 슬롯 터치 → 시간 단축 팝업

**13. 피냐타(유실물) 선택 팝업**
- 보상 리스트 탭
- 대체 보상 탭
- 보상 확률 표시
- 피냐타 슬롯 3개 (자동 선택)
- 미보유 시 토스트
- 선택 버튼

**14. 시간 단축권(달빛 정수/꿈나라 포션) 팝업**
- 셀 터치 진입
- 사용 횟수 슬라이더 (-/+/MAX)
- 사용하기 버튼
- 보유 0개 시 처리
- 10분 단위 차감
- 사용 후 슬롯 상태 갱신

**15. 보상 획득 팝업**
- SlotOpen 트윈 애니메이션
- 획득 완료 팝업 (보상 아이템 리스트)
- 확인하기 버튼

**16. 유실물 게시판/주간 유실물 미션**
- 진행도 표시 (123/250)
- 유실물 카드 그리드
- 진행도 보상 게이지 (50/100/150/200)
- 대체 보상 안내

**17. 주간 꿈세계 스탬프 미션**
- 스탬프 5종 진행도
- 미션 팝업 (성좌/별자리 맵)
- 미션 노드 (개별 진행도)
- 초기화 카운트다운 (수요일 KST 9:00)
- 완료 보너스
- 보상 수령
- 미션 종류 10종 (특정 쿠키/무작위/별석탄/약초/몬스터 처치/점수/드래프트/소생 등)

**18. 햄버거 메뉴**
- 다른 콘텐츠 이동 시 버그 검증

1. **슬롯 상태 5종(NoItem/Empty/InProgress/Complete/SlotOpen) 분기**: 각 상태별 UI 표기, 알림 닷 종류, 터치 시 동작이 모두 다르므로 상태별로 명확히 분리하여 작성한다.
