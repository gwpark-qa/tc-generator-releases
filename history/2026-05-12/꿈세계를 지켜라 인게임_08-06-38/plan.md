## 1단계: TC 설계

### 기능 목록

**1. 인게임 진입 및 구성**
- 라운드 진입 (총 3 라운드 구성: 라운드 1·2 일반 스테이지, 라운드 3 보스 스테이지)
- 인게임 진입 시 전체 인게임 UI 출력
- 캐릭터/영역 구성 (플레이어 쿠키, 헬리어, 별석탄, 약초더미, 모닥불, 몬스터, 보상상자 등)

**2. 플레이어 - 쿠키 / 헬리어**
- 헬리 모드(헬리어 탑승) 신규 모드
- 쿠키 스킬: 일반 공격 / 차지 게이지 축적 / 차지 공격 발동
- 헬리어 연료 소진 시 동작
- 헬리어 탑승 / 강제 하차 / 헬리어 파괴 처리

**3. 인게임 규칙**
- 클리어 조건: 라운드 내 모든 몬스터 처치 / 보스 처치
- 실패 조건: 헬리어 파괴 또는 쿠키 사망
- 일시정지 시 동작

**4. 인게임 HUD / UI**
- 별가루 카운터 (우상단)
- 연료 게이지 (헬리어 잔여 연료 시각화, 1초당 1 소진)
- 연료 50%/20% 도달 시 색상 변경 (노란색/빨간색)
- 경험치 프로그레스 바 (헬리어 레벨업 게이지)
- 미니맵
- 캐릭터 아이콘
- 카운트다운(타이머)

**5. 라운드 시작 투표 UI**
- 본인/타플레이어 카운트 표시
- 준비 완료 노출
- 모든 플레이어 동의 시 라운드 시작

**6. 드래프트(로그라이크 선택지)**
- 드래프트 진입 (라운드 진입 시 / 레벨업 시)
- 메인 드래프트 선택 (3개 카드, 각 카드 효과 표시)
- 서브 드래프트 선택 (난이도에 따라 추가 등장, 3개 카드)
- 카드 NEW 표기
- 선택 후 효과 적용
- 드래프트 카드 종류(발화 이언/직격폭화/들끓는 의지/처치 의지/관통벽 등 9종)

**7. 인게임 기믹 - 별석탄 광석 / 별가루**
- 별석탄 광석 기본 상태 출력 (반짝거리는 이펙트)
- 별석탄 채굴 (다가가서 상호작용 시 채굴)
- 별석탄 광석 파괴 (총 N회 채굴 시 파괴)
- 별석탄/별가루 드랍 & 아이템 모션
- 별가루 획득 시 카운트 증가
- 별가루로 헬리어 연료 게이지 충전

**8. 인게임 기믹 - 약초 더미 / 약초**
- 약초 더미 기본 상태 출력
- 약초 채집 (3회 채집 시 약초 1개 드랍)
- 약초 더미 파괴
- 약초 드랍 & 아이템 모션
- 약초 획득 시 쿠키 체력 회복

**9. 보상 상자 & 달빛 포션**
- 몬스터 처치 시 빛 구체 포물선 드랍 애니메이션
- 빛 구체 꼬리 궤적 이펙트
- 떨어진 지점에 아이템 등장
- 아이템이 바닥에서 둥둥 떠있는 애니
- 빛 기둥 이펙트
- 보상 상자 / 달빛 포션 획득

**10. 몬스터 / 적**
- 일반 몬스터 출현/처치
- 엘리트 몬스터 출현/처치 (큰 비중)
- 상급 몬스터 출현/처치
- 보스 몬스터 출현/처치
- 몬스터 웨이브 알림 ("적들이 몰려옵니다!")
- 몬스터 처치 시 경험치 / 드랍 아이템 지급

**11. 레벨업 (경험치)**
- 몬스터 처치 시 경험치 게이지 증가
- 게이지 풀 시 헬리어 레벨업
- 레벨업 시 Level Up 텍스트 표시
- 레벨업 시 서브 드래프트 선택 창 등장

**12. 인게임 카메라**
- 별석(영역) 중심의 카메라 추적
- 카메라가 영역 경계에 닿으면 정지
- 쿠키가 영역을 끌고가는 경우 카메라 이동
- 쿠키가 카메라 밖으로 빠져나갈 수 없음(외곽 방치)
- 멀티플레이어: 각 쿠키별 영역(카메라) 분리

**13. 인게임 플로우 (라운드 진행)**
- 진입 → 드래프트 선택 → 플레이 시작 → 라운드 종료 → 다음 라운드 진입 → 반복
- 라운드 종료 시 COMPLETED 표시
- 라운드 종료 점수 표시
- 다음 라운드 진입 (서브 드래프트 등장)

**14. 일시정지**
- 일시정지 진입 (본인/타플레이어 카운트)
- 종료 버튼

**15. 난이도별 변경점**
- 쉬움 / 보통(+1 위험) / 어려움(+2 위험)
- 몬스터 수/난이도 변경
- 메인 드래프트 카드 차이
- 위협 요소(설계 가지) 등장 여부
- 레벨업 수 차이
- 연료 등급 변경

**16. 랭킹 / 점수 집계**
- 라운드별 클리어 시간 점수 환산 (180,000ms 기준, 1ms당 약 1.67점)
- 난이도 가중치 적용 (쉬움 0% / 보통 +30% / 어려움 +50%)
- 라운드별 점수 합산 (최대 1,200,000점)
- 라운드 종료 후 점수 표시

### 중점 확인 사항

1. **헬리어 연료 시스템**: 연료가 게임의 핵심 메커니즘 — 1초당 1 소진, 별가루로 충전, 50%/20%에서 색상 변경되는 조건 명확히 분리해서 TC화. 연료 0 도달 시 헬리어 동작 정지 / 사망 → 게임 실패 (Critical)
2. **드래프트(로그라이크) 시스템**: 메인/서브 드래프트가 별도로 존재하고 등장 조건이 다름(라운드 진입 vs 레벨업). 카드 종류 9가지가 각각 다른 효과를 가지므로 효과 적용 TC를 카드별로 분리
3. **점수 집계 로직**: 시간 기반 점수 환산(1ms당 약 1.67점), 난이도별 가중치(쉬움 0% / 보통 30% / 어려움 50%), 최대 1,200,000점 — 데이터 손실 가능성이 있는 영역이므로 Major/Critical 부여
4. **별석탄 / 약초 채굴 사이클**: 기본 상태 → 채굴(채집) → 사이즈 단계별 변화 → 파괴 → 드랍 — 일련의 흐름을 sub3로 세분화하여 각 단계 별 TC 작성
5. **카메라 추적 로직**: 별석 중심 + 쿠키 영역 이탈 방지 + 멀티플레이 시 각 쿠키별 카메라 분리는 단순 시각 외에 게임 진행에 영향을 주므로 별도 TC 분리
6. **난이도별 분기**: 쉬움/보통/어려움에 따라 몬스터 수, 드래프트 카드 수, 위협 요소, 레벨업 수가 달라짐 — 각각 분리하여 작성
7. **라운드 진행 플로우**: 3개 라운드 구조(라운드 1·2 일반, 라운드 3 보스)와 라운드 간 서브 드래프트 등장 흐름
8. **알림/UX**: "적들이 몰려옵니다!" 알림, COMPLETED 표시, Level Up 텍스트, NEW 카드 표시 등 화면 출력 케이스
