### 기능 목록

**1. 이벤트 진입**
- 메인 로비 노티 배너 표시
- 이벤트 사이드 패널 (이벤트 02/03/04 항목 리스트)
- 반죽 게임즈 테스트 센터 진입 썸네일
- 이벤트 메인 화면 (잔여 시간 표시 22일 16시간, 정보(i) 버튼, 뒤로가기)
- 이벤트 메뉴 (튜토리얼/랭킹/미션 버튼)
- 일일 보상 목표 점수 표시 (Normal/Complete/Done)
- START 버튼

**2. 미니게임 인게임 UI**
- 메뉴 버튼 (좌상단)
- Wave 표시 + 다음 웨이브까지 남은 게이지바
- Best/Score 표시
- 시작 지점 (구슬 스폰 위치)
- 종료 지점 (게임오버 트리거)
- 레일/구슬 (6색 구슬, 둥근 레일)
- 포대 (버블껌 대포) - 발사 방향 전환, 발사 궤적 표시
- 구슬 교체 (현재/다음 구슬, 포대 주변 터치)
- 스킬 - 일시정지 (1회/판)
- 스킬 - 같은 색 구슬 제거 (1회/판)
- 피버샷 게이지 (충전, 만충, 강화 발동)
- 보너스 점수 별(★) 스폰/타격/사라짐
- START 안내문, WAVE 안내문 (오버랩)

**3. 튜토리얼**
- 튜토리얼 팝업 진입 (가이드 아이콘 터치)
- 일시정지/딤드 처리
- 페이지 인디케이터 도트, 페이지 이동 화살표
- 튜토리얼 6개 챕터 (발사/구슬 교체/콤보/피버/스킬/종료 조건)

**4. 메뉴 (플로팅 목록)**
- 메뉴 버튼 터치 시 플로팅 노출
- 일시정지/딤드 처리
- 설정/미션/랭킹/가이드/나가기 항목

**5. 설정 팝업**
- 사운드/효과음/진동 토글
- 나가기/다시하기/계속하기 버튼
- 닫기(X) 버튼

**6. 다시하기 확인 팝업**
- 진행도 초기화 안내 문구
- 취소/확인 버튼
- 닫기(X) 버튼

**7. 미션 팝업**
- 종료까지 남은 시간 표시
- 미션 리스트 (진행도, 보상, 획득 버튼)
- 진행 중/달성/완료 상태별 분기
- 보상 수령 동작
- 닫기(X) 버튼

**8. 랭킹 팝업**
- 종료까지 남은 시간 표시
- 랭킹 리스트 (1/2/3위 메달, 플레이어, 점수)
- 본인 행 강조 표시
- 닫기(X) 버튼

**9. 게임 종료**
- 구슬이 레일 끝 도달 시 게임오버 트리거
- 남은 구슬 빨려 들어가는 연출
- 결과 팝업 (BEST SCORE / SCORE / New Record / 나가기 / 다시하기)

**10. 이펙트/연출**
- 콤보 텍스트 트윈
- 구슬 획득 이펙트
- 피버 게이지 파티클 충전 연출
- FEVER 텍스트 흔들림 트윈
- WAVE 안내 텍스트 연출
- 자석(같은 색 제거) 등장/끌어당김 이펙트
- 일시정지 화면 가장자리 알림 이펙트
- 일시정지 시계바늘 회전 이펙트
- FEVER 발동 포대/특별구슬 연출
- 피버 게이지 중앙 파티클 반짝임

### 중점 확인 사항

1. **카테고리 통합 원칙 준수**: 미니게임 인게임은 메인 카테고리로 두되, 이벤트 진입은 별도 main으로 분리하여 사용자 플로우 순서대로 배치한다. (이벤트 → 미니게임 → 메뉴/설정/미션/랭킹 → 종료)

2. **단일 기대결과 원칙**: 한 TC에 두 개 이상의 기대결과가 들어가지 않도록 분리. 예) "딤드 처리되며 팝업이 출력된다" → "딤드 처리된다" / "팝업이 출력된다" 분리

3. **스킬 1회 사용 제한**: 일시정지/같은 색 구슬 제거 스킬은 게임당 1회 사용 제한 → 사용 후 비활성/딤드 처리 케이스 필수

4. **피버샷 강화 상태 취소 불가**: 강화 진입 후 취소 불가 사양을 별도 TC로 명시

5. **결과 팝업 New Record 분기**: 최고 점수 갱신/미갱신 두 분기 모두 TC 작성

6. **미션 획득 버튼 상태 분기**: 진행 중(비활성) / 달성(활성) / 수령 후(비활성) 3가지 상태 분리 작성

7. **이펙트/연출 케이스 별도 작성**: 콤보 트윈, 자석 등장/끌어당김, FEVER 텍스트 흔들림, 화면 가장자리 알림, 시계바늘 회전 등은 Normal 우선순위로 별도 TC 작성

8. **딤드/일시정지 처리**: 메뉴/설정/미션/랭킹/튜토리얼 팝업 진입 시 공통적으로 게임 일시정지 및 딤드 처리 필요 → 각 진입점별로 누락 없이 작성

9. **인게임 진입 후 TC는 작성하지 않음**: START 버튼 터치 → 미니게임 진입까지만 작성 (단, 인게임 UI 자체는 기획서에 명시되어 있어 UI 출력 케이스로 작성)

10. **예외 케이스 위치**: 각 카테고리 마지막에 배치 (예: 게임오버 시 미션 진행도/랭킹 갱신 관련)

11. **수치 미정 항목 처리**: "일시정지 N초", "별 N초 후 사라짐", "보너스 점수 스폰 N초" 등 미정 수치는 "n초"로 그대로 표기하지 않고 "일정 시간" 등으로 풀어서 서술

12. **화면 이동 케이스 위치**: 각 카테고리 마지막에 배치 (예: 나가기 → 이벤트 화면 이동, START → 미니게임 진입)
