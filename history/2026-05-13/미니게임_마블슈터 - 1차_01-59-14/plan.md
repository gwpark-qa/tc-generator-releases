### 기능 목록

**1. 진입 / 이벤트 모드 진입**
- 인게임 메인 로비 노티 배너 탭을 통한 이벤트 진입
- 이벤트 사이드 패널에서 "반죽 게임즈 테스트 센터" 선택
- 이벤트 메인 화면(잔여 시간, 정보 i 버튼, 뒤로가기, 이벤트 카테고리 탭) 노출
- 메인 비주얼/캐릭터 일러스트, 튜토리얼/랭킹/미션 버튼, START 버튼
- 이벤트 모드 썸네일(모니터 연출 - MINI GAME / Press the button)
- 일일 보상 목표 점수 UI 상태 (Normal/Complete/Done)

**2. 튜토리얼**
- 가이드 아이콘 터치로 튜토리얼 팝업 진입
- 게임 화면 일시정지 및 딤드 처리
- 튜토리얼 팝업 구성(타이틀, 설명, 이미지, 페이지 인디케이터 도트 3개, 화살표 `>`)
- 튜토리얼 6단계 컨텐츠 (발사 / 구슬 교체 / 콤보 / 피버 / 스킬 / 종료 조건)
- 페이지 좌/우 이동
- 팝업 닫기

**3. 이벤트 모드 인게임 UI**
- 메뉴 버튼 (1-A) 터치 시 플로팅 목록 노출 + 일시정지/딤드
- 플로팅 목록(설정/미션/랭킹/가이드/나가기) (1-B)
- 현재 웨이브 표시 + 다음 웨이브까지 게이지바
- Best/Score 표시
- 시작 지점(우상단 구슬 스폰) / 종료 지점(레일 끝)
- 포대(버블껌 대포) - 중앙 위치, 방향 전환, 발사 궤적
- 구슬 교체 (현재/다음 구슬, 포대 주변 터치)
- 레일 및 구슬 (둥근 형태, 6색 구슬)
- 스킬(일시정지 / 같은 색 구슬 제거) - 1회 사용 제한
- 피버샷 게이지 (충전/만충/강화/취소 불가)
- 보너스 점수(별 ★) - 랜덤 스폰, 직접 타격, n초 후 사라짐
- WAVE 안내 텍스트
- START! 인디케이터

**4. 이벤트 모드 상세 / 이펙트**
- 콤보 적용 후 피버 게이지 차오름 (콤보 텍스트 트윈, 구슬 획득 이펙트, 게이지 충전 파티클)
- FEVER 발동 (FEVER 텍스트 흔들림, 포탄 변경, 특별한 공 난사, 게이지 중앙 파티클)
- WAVE 안내 오버랩 (Start 안내문 동일 처리)
- 일시정지 아이템 (화면 가장자리 알림 이펙트, 시계바늘 회전)
- 같은 색 구슬 제거 (자석 등장 이펙트, 끌어당김 이펙트)

**5. 게임 종료**
- 한 개 구슬이라도 레일 끝 도달 시 게임 종료
- 잔여 구슬 구멍으로 빨려 들어가는 연출
- 게임 화면 딤드 처리
- 결과 팝업(테스트 결과/RESULT) - New Record, BEST SCORE, SCORE
- 최고 점수 갱신/미갱신 분기
- 나가기 / 다시하기 버튼

**6. 설정**
- 설정 아이콘 터치 시 팝업 출력, 게임 일시정지/딤드
- 미니 게임 설정 팝업 (X 닫기 버튼)
- 사운드 / 효과음 / 진동 토글 (활성/비활성)
- 나가기 / 다시하기 / 계속하기 버튼
- 다시하기 확인 팝업 (취소/확인 2버튼, X 닫기)
- 진행도 초기화 후 게임 재시작

**7. 미션**
- 미션 아이콘 터치 시 미션 팝업 출력, 게임 정지/딤드
- 종료까지 남은 시간 표시 (시계 아이콘 + 카운트다운)
- 미션 목록(라벨/미션명/진행 게이지/보상 아이콘/획득 버튼)
- 진행도 미달성 시 획득 버튼 비활성
- 진행도 100% 달성 시 획득 버튼 활성
- 보상 수령 후 완료 처리
- X 닫기

**8. 랭킹**
- 랭킹 아이콘 터치 시 랭킹 팝업 출력, 게임 정지/딤드
- 종료까지 남은 시간 표시
- 헤더(랭크/플레이어/점수)
- 랭킹 정보 리스트 (1~3위 메달 아이콘, 플레이어 아이콘)
- 본인 랭킹 행 강조
- X 닫기

### 중점 확인 사항

1. **이미지 분석 결과 매칭**: 녹색 사각형/동그라미 + 숫자 강조 영역과 우측 설명을 1:1로 매칭하여, 단순 영역 출력 케이스(예: "포대 영역 출력") + 해당 영역의 동작 케이스(예: "발사 궤적 표시")를 분리 작성.

2. **이펙트/연출 케이스 누락 방지**: 기획서에 명시된 모든 이펙트를 별도 TC로 작성
   - 콤보 텍스트 트윈, 구슬 획득 이펙트, 피버 게이지 충전 파티클
   - FEVER 텍스트 흔들림 트윈, 게이지 중앙 반짝이 파티클
   - WAVE 안내 텍스트(START 안내문과 동일 처리)
   - 자석 등장(우물쭈물) 이펙트, 끌어당김 이펙트
   - 일시정지 시 화면 가장자리 알림 이펙트, 시계바늘 회전 이펙트
   - 잔여 구슬 빨려 들어가는 게임오버 연출

3. **1회 사용 제한 스킬**: 일시정지 / 같은 색 구슬 제거 스킬은 게임당 1회만 사용 가능 → 사용 후 비활성/딤드 처리 케이스 분리 작성.

4. **피버 강화 상태**: 게이지 만충 시 터치 → 강화 상태 진입 → 진입 후 취소 불가 케이스를 별도 TC로 분리.

5. **결과 팝업 분기**: New Record 라벨 표시 분기(최고 점수 갱신 시 표시 / 미갱신 시 미표시)를 두 개의 별도 TC로 작성.

6. **다시하기 확인 팝업**: 설정 내 다시하기 → 진행도 초기화 확인 팝업 → 확인/취소 분기를 각각 분리하여 작성.

7. **미션 진행도 분기**: 진행도 미달성/달성/수령 완료의 3가지 상태에 따른 획득 버튼 상태 변화를 각각 별도 TC로 작성. (기획서상 진행 중 미션도 활성으로 보이는 부분은 일반 기획 스펙 기준으로 작성)

8. **랭킹/미션 잔여 시간**: 시계 아이콘 + 카운트다운 출력(예: "20일 2시간") 표시 케이스 작성.

9. **카테고리 통합 원칙 준수**: 설정/미션/랭킹/튜토리얼은 인게임 메뉴에서 진입하지만, 각각 독립된 기능 영역이므로 별도 main 카테고리로 분리하되, 인게임 메뉴 플로팅에서 각 항목 터치 시 해당 팝업이 출력되는 흐름 TC는 "이벤트 모드 인게임 UI > 메뉴" 하위에 배치하고, 각 팝업의 세부 기능은 별도 main 카테고리에서 다룸.

10. **인게임 진입 이후 케이스 제외**: START 버튼 터치 후 인게임 진입은 작성하되, 실제 인게임 플레이 자체에 대한 동작은 작성하지 않음. 단, 인게임 UI 출력/버튼 동작/이펙트 등은 클라이언트 UI 기능이므로 작성 대상에 포함.

11. **문체 규칙 준수**: "~시,", 수동형, "정상" 표현 금지 등을 모든 TC에 일관되게 적용.
