**레이드 인게임**

- 입장 정보 및 Play 버튼

### 중점 확인 사항

1. **화면 전환 케이스**: 각 단계별 화면 전환이 누락되지 않도록 순서대로 작성 (총 10개 주요 전환 케이스 확인)

2. **조건부 동작 검증**: 
   - 입장 전투력 미달 시 비활성화 + 토스트 (`equipment.dungeon.not.host`)
   - 이전 난이도 미클리어 시 비활성화 + 토스트 (`single_boss_challenge.locked`)
   - 잠금 보스/난이도 터치 시 안내 팝업

3. **수치 변경사항 확인**:
   - 티켓 입출: 5장 → 2장 변경
   - 배수 설정: 기본 3장, 패스 구매시 5장
   - 멀티플레이: 4인 → 2인 변경
   - 드랍 연출: 3초 지속

4. **기능 제거 검증**:
   - 코코방울 연출/축복/포인트 완전 제거
   - 트로피 시스템/상자/포인트 완전 제거  
   - 무지개 상자 이벤트 제거
   - 랜덤 상자 기능 제거
   - 다시까기 기능 제거 (배수로 대체)

5. **팝업 및 토글 동작**:
   - 난이도 변경 팝업 중복 클릭시 토글 닫힘
   - 각종 상세 정보 팝업 출력/닫기
   - 싱글/멀티 토글 Default 상태 및 전환

6. **아이템 표시 시스템**:
   - 12종 아이템 최적화 표시
   - 초과시 스크롤 기능
   - 아이템별 분류 (장비, 항아리, 설계도, 조각 등)
