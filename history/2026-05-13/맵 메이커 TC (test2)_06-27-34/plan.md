### 기능 목록

기획서에서 파악한 모든 기능을 번호(오름차순) 기준으로 정리합니다.

**[0] 심사하기 진입 화면**
- 추천 맵 3장 카드 출력 (맵 이름, 제작자, 등록일, 심사하기 버튼)
- 카운트다운(맵 변경까지 남은 시간) 출력 및 자동 갱신
- [맵 변경] 버튼 → 변경 팝업 호출
- 닫기(X) 버튼

**[1] 맵 메이커 허브**
- 상단 인디케이터 (보석, 골드, 채팅, 설정, 홈, 메뉴)
- 메인 메뉴 3종: 맵 만들기 / 심사하기 / 찾아보기
- 보조 탭: 맵 랭킹 / 제작자 랭킹
- 뒤로가기 동작

**[2] 나의 맵 - 보유 상태**
- 슬롯 카운트 (n/2) 표기
- 정렬 드롭다운
- 보유 맵 카드 (코드, 제작자, 맵 이름, ♥/👤, 제작일, 복사, 편집하기)
- [슬롯 추가!] 버튼 동작

**[3] 나의 맵 - 빈 상태**
- (0/2) 슬롯 카운트
- [새로 만들기] 버튼 동작

**[4] 맵 생성하기 입력 (나의 맵 → 진입)**
- 맵 이름 입력 (0/15)
- 맵 설명 입력 (0/80)
- 맵 테마 [선택하기] 진입
- [생성하기] 버튼 활성/비활성
- 금칙어/특수문자/이모지 입력 예외
- 백버튼/X 시 변경사항 경고

**[5] 맵 테마 선택 - 보유 (맵 생성하기 → 진입)**
- 테마 카드 그리드 출력 (썸네일, 명칭, 난이도 라벨, SAMPLE 배너)
- ⓘ 정보 아이콘 (테마 설명 팝업)
- [사용하기] 버튼 동작 + 테마 변경 시 전용 오브젝트 삭제 경고
- 하단 안내문 출력

**[6] 맵 테마 선택 - 잠금 (맵 생성하기 → 진입)**
- 잠금 상태(자물쇠) 표시
- [젬 교환소 구매] 버튼 동작 → 젬 교환소 이동

**[7] 찾아보기 - 오늘의 추천 맵**
- 탭 전환 (오늘의 추천 맵 / 코드로 찾기)
- 정렬 드롭다운 (최신순 등)
- 자정 갱신 안내문
- 맵 카드 (코드, 썸네일, ♥/👤, 제작자, 제작일, 맵 이름, 달려보기 버튼)
- 코드 복사 아이콘
- [달려보기] 버튼 동작

**[8] 찾아보기 - 코드로 찾기**
- 코드 입력 필드, ❌ 초기화 버튼
- [찾기] 버튼 동작
- 빈 입력/형식 오류/미존재/비공개 코드 예외

**[9] 찾아보기 빈/로딩/에러 상태**
- 로딩/빈/에러 안내

**[10] 맵 변경 팝업 (심사)**
- 다이아 100 차감
- 변경하기 버튼
- 카운트다운 유지 규칙
- 보석 부족 케이스

**[11] 맵 랭킹 (Top 100)**
- 시즌 정산 카운트다운
- Top 100 특별 보상 탭
- 이달의 맵 TOP 100 리스트 (순위, 맵명, ♥, 🍪, 제작자, 칭호, 썸네일 ▶)
- 재생(▶) 진입
- GO TO TOP 동작

**[12] 심사하기 상세 (PLAY) 팝업**
- 맵 정보 출력 (썸네일, 이름, 제작자, 맵 코드, 등록일)
- 맵 설명 출력
- 맵 코드 📋 복사
- 조합 고정 / 능력 OFF 옵션 라벨
- [PLAY!] 버튼 → 심사 모드 진입
- 잘못된 맵 / 삭제된 맵 예외

**[13] 제작자 랭킹 (Top 100)**
- 시즌 정산 카운트다운
- 1위 트로피 카드 (캐릭터, 말풍선, 칭호, 닉네임, 길드)
- 내 순위 행 (기록 없음 ❤0)
- Top 100 리스트
- GO TO TOP 동작

**[14] 제작자 랭킹 빈 상태**
- 1위 공석 카드
- "가장 먼저 1등이 되어보세요!" 안내
- 첫 등록 시 13번 화면 전환

**[15] 맵 랭킹 빈 상태**
- 시즌 정산 카운트다운 유지
- "가장 먼저 1등이 되어보세요!" 안내

**[16] 슬롯 구매 팝업**
- 보석 잔액 출력
- 슬롯 미리보기
- [💎 3,000 구매하기] 버튼
- 보석 부족 / 최대 슬롯 도달 예외
- 네트워크 오류 시 차감-슬롯 불일치 처리

> ※ 사용자 피드백 반영:
> - 기존 [2]/[3]/[14] 맵 테마 선택(보유/잠금) 및 맵 생성하기 입력 카테고리를 "나의 맵" 산하의 하위 카테고리로 통합.

### 중점 확인 사항

1. **재화 차감/롤백 무결성 (Critical 핵심)**
   - 슬롯 구매(💎 3,000): 차감 후 슬롯 미증가 시 재처리, 보석 부족 시 비활성 처리 (나의 맵 > 슬롯 구매 팝업)
   - 맵 변경(💎 100): 차감 후 추천 맵 미갱신 케이스, 보석 정확히 100 보유 시 동작

2. **카운트다운 타이머 동작**
   - 추천 맵 변경([0]): 시간 만료 직전 [심사하기] 누른 경우 우선순위
   - 시즌 정산(맵/제작자 랭킹 및 각 빈 상태): 0 도달 시 랭킹 마감 및 보상 지급
   - 맵 변경 후 카운트다운 유지 규칙([10])

3. **빈 상태 ↔ 데이터 상태 전환**
   - 제작자 랭킹 빈 상태 → Top 100 (첫 등록 시 즉시 전환)
   - 맵 랭킹 빈 상태 → Top 100 (맵 첫 등록 시 즉시 전환)
   - 나의 맵 빈 상태 → 보유 상태 (슬롯 첫 사용 시 카운트 표기 갱신)

4. **나의 맵 → 맵 생성하기 → 맵 테마 선택 / 슬롯 구매 팝업 연계 (통합 카테고리)**
   - 빈 상태 [새로 만들기] / 보유 상태에서 맵 생성하기 진입 경로 일관성
   - 맵 이름 15자 / 맵 설명 80자 초과 차단, 금칙어·특수문자·이모지 입력 처리
   - 백버튼/X 시 미저장 변경사항 경고
   - 맵 테마 선택 시 보유 테마 적용 + 기존 테마 전용 오브젝트 삭제 경고 팝업
   - 잠금 테마 [젬 교환소 구매] → 젬 교환소 이동 분기
   - SAMPLE 배너 카드 사용 시 복제 vs 신규 생성 분기
   - 보유 상태 [슬롯 추가!] → 슬롯 구매 팝업 호출 → 보석 차감 / 슬롯 1개 증가 / 최대 슬롯 도달 시 비활성

5. **조합 고정 / 능력 OFF (심사 상세 PLAY)**
   - 심사 모드 진입 시 강제 적용 검증
   - 보유하지 않은 쿠키 사용 시 처리 정책

6. **맵 코드 복사 및 검색 (코드로 찾기/심사 상세/나의 맵/추천 맵)**
   - 클립보드 복사 + 토스트 출력
   - 빈 입력 / 형식 오류 / 미존재 / 비공개·삭제된 맵 코드 예외

7. **랭킹 정렬 및 동점 처리 (맵/제작자 랭킹)**
   - ❤ 동점 시 2차 정렬 키
   - GO TO TOP 동작
   - 본인이 만든 맵에 ❤ 부여 가능 여부

8. **다국어 표시 (제작자 랭킹)**
   - 일본어/중국어 닉네임·칭호·길드명 깨짐 여부

9. **네트워크 예외**
   - 추천 맵 로딩 실패, 슬롯 구매 중 단절(나의 맵 > 슬롯 구매 팝업), 찾아보기 통신 오류
   - 재시도 UI 노출 여부
