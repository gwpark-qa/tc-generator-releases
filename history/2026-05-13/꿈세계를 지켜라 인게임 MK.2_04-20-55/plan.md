### 기능 목록

**1. 쿠키 선택 및 매칭**
- 전용 쿠키 10마리 + 시크릿(달빛술사) 랜덤 풀
- 스타트 쿠키 / 해금 쿠키 / 2주년 차 업데이트 추가 쿠키(스타더스트, 밀키웨이)
- 중복 픽 매칭 처리 (기본 불가, 매칭 안 잡힐 시 허용)
- 랜덤 선택 시 1라운드 시작 시 서브 드래프트 1개 제공
- 시크릿 쿠키 확률적 등장

**2. 인게임 진입 / 라운드 구조**
- 3라운드 구성 (라운드당 100초 내외, 총 5분)
- 트리 구조 난이도 분기 (쉬움/중간/어려움)
- 라운드 진도 UI (지도 노드)
- 클리어/실패 조건 (열차가 맵 끝 도달 / 열차 체력 0 / 전원 사망)

**3. 인게임 HUD UI**
- 타이머 (좌측 상단)
- 캐릭터 아이콘 + HP 게이지 (상단 중앙)
- 연료/경험치 프로그레스 바 (상단 중앙)
- 스코어 UI (우측 상단)
- 일시정지 버튼
- 위치 인디케이터 (시야 밖 요소)
- 라운드 진도 현황 UI

**4. 인게임 조작 UI**
- 이동 조이스틱
- 공격 버튼
- 특수 스킬 버튼
- 궁극기 버튼
- 대시 버튼
- 이모티콘 버튼
- 쿠키별 조작 UI 변동

**5. 꿈열차 (운영 오브젝트)**
- 맵 중앙 가로지르며 전진
- 별석탄 투입 → 연료 게이지 충전
- 약초 투입 → 회복 영역 생성
- 연료 게이지 자연 감소
- 연료 최대치 초과 시 1초 유지 → 감소
- 연료 50% 노란색 / 20% 빨간색 + 부스트 이펙트
- 연료 0 도달 시 열차 정지 → 체력 감소 → 파괴

**6. 별석탄 광석 기믹**
- 작은 사이즈 (3×3m) / 큰 사이즈 (6×3m)
- 캐기 버튼 (곡괭이 모양 변경)
- 보호막 개수만큼 타격
- 캐기 애니메이션 중 무방비
- 홀드 시 끊김없이 채굴
- 부서지면 별석탄 n개 드랍
- 드랍 위치 보정 (벽이 아닌 위치)

**7. 별석탄 (휴대 아이템)**
- 별사탕 모양, 폭탄형
- 들어올리기 (공격으로 상호작용)
- 들었을 때 다른 조작 비활성화 (X 표시)
- 던지기 (스킬, 포물선 궤적)
- 놓기 (공격, 바닥에 떨굼)
- 피격 시 자동 낙하
- N초 후 3초 깜빡임 → 소멸
- 다시 들었다 놓으면 카운트 리셋
- 열차에 투입 시 연료 충전

**8. 약초 더미 기믹**
- 3×3m 사각형 풀 더미
- 3회 타격 시 파괴
- 타격 시 진동 피드백
- 파괴 시 약초 1개 드랍
- 드랍 위치 보정

**9. 약초 (휴대 아이템)**
- 별석탄과 유사 휴대 구조
- 들기/던지기/놓기/피격 낙하
- 열차에 넣으면 열차 중심 원형 회복 영역 (n초 지속)
- 자연 소멸 (N초 + 3초 깜빡임)

**10. 몬스터**
- 일반/엘리트/보스
- 일반 몬스터 처치 → 젤리빈 드랍
- 보스 (3라운드 등장)
- 엘리트 버프

**11. 보상 상자 / 달빛 포션 (드랍)**
- 몬스터 처치 시 빛 구체 포물선 드랍
- 빛 기둥 이펙트
- 난이도에 따라 등급 차이

**12. 경험치 / 레벨업 (공용)**
- 젤리빈 획득 → 공용 경험치
- 총 용량 100, 젤리빈 1개당 N%
- 최대 Lv 15
- 레벨업 시 "Level Up!" 텍스트 + 이펙트
- 서브 드래프트 3개 등장

**13. 플레이어 사망 / 유령 / 소생**
- 사망 시 유령 변환
- 유령 상태 힐 불가
- 팀원이 n초 상호작용 시 소생

**14. 드래프트 - 메인 선택지**
- 2~3라운드 시작 시 3개 등장
- 카드 UI (타입 아이콘, 무기 이미지, 효과 설명, 태그)
- NEW 배지 (미선택 시)
- 중복 픽 불가
- 새로고침 (기본 3회, 직전/획득 효과 제외)
- 조건부 카드 (달빛술사 무기, 스타더스트, 밀키웨이)

**15. 드래프트 - 서브 선택지**
- 3개 등장
- 메인과 크기 동일, 타입 아이콘/태그 제거, 배경색 차이, 스탯 아이콘
- 젤리빈 레벨업 / 난이도 상승 / 1라운드 랜덤픽 시 등장
- 태그 3번 중복 획득 시 메인 강화 (최대 1회)

**16. 드래프트 카드 효과 (테스크 계열)**
- 발화 석탄 / 척척폭폭! / 불굴의 의지 / 저리 비켜! / 찌릿찌릿 / 함정 투척 / 전설의 약초 / 레이저빔 / 고속채굴

**17. 일시정지 UI**
- 팀원의 선택 효과 표시
- 강화된 효과 시각적 차별화
- 중단(나가기) → 포기 투표 팝업
- 팀모이기 버튼

**18. 라운드 클리어 / 난이도 투표**
- 라운드 클리어 화면 → 점수 정산
- 3개 난이도 선택지
- 무작위 1인 선정 진행
- 1명 진입 시 10초 카운트다운
- 3명 진입 시 즉시 투표
- 시간 만료 + 미선택 시 가장 쉬운 난이도
- 인게임 시간 정지
- 카메라 당김

**19. 카메라**
- 열차 중심 따라오기
- 영역 경계 도달 시 카메라 멈춤
- 유저별 고유 영역

**20. 점수 계산 / 랭킹**
- 남은 시간 ms 기반 (1ms ≒ 1.67점)
- 3분 완벽 시 300,000점
- 소수점 버림
- 난이도 가중치 (쉬움 0%, 보통 30%, 어려움 50%)
- 최대 1,200,000점
- 라운드별 누적

**21. 규약 추가 (중간/어려움)**
- 회오리 / 시야 / 폭탄 등 랜덤 규약

**22. 문 기믹 (중간/어려움)**
- 레버 상호작용으로 문 개방

**23. 예외 처리**
- 다수결 동률
- 즉사 피격 시 별석탄 처리
- 깜빡임 중 픽업 가능 여부
- 무적 상태 피격 시 드롭 여부
- 네트워크 단절
