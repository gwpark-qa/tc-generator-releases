# TC 설계: 랜덤챌린지 - 주사위 시즌

### 기능 목록

기획서에서 파악한 모든 기능을 나열합니다.

#### 주사위 획득
1. **최초 지급**: 랜덤 챌린지 [입장하기] 시 일반 주사위 N개 1회 지급
2. **최초 지급 연출**: "누군가 말을 걸어왔다!" 연출 + 쿠키 접근 + 시즌 버프 안내
3. **시즌 버프**: 획득 점수 n점마다 일반 주사위 n개 획득 (주자별 최고 점수 기준 1회)
4. **주자 선택 시 지급**: 주자에 할당된 주사위 즉시 지급 (별도 획득 연출 없음)
5. **주자별 주사위 배치**: 최소 1개, 낮은 확률로 2개 배치
6. **주자 주사위 종류**: 일반 주사위(랜덤 상태) + 굴려진 주사위(숫자 확정 상태)
7. **플레이 보상**: 결과 화면에서 최고 점수 기준 주사위 수량 산정 및 노출
8. **플레이 보상 지급 시점**: 다음 주자로 [이어달리기] 진행 시 지급
9. **플레이 보상 지급 연출**: "누군가 말을 걸어왔다!" 연출과 주사위 지급
10. **재도전 시 점수 처리**: 이전 점수보다 낮으면 주사위 수량 변동 없음, 높으면 갱신
11. **스페셜 이벤트**: 랜덤/특정 주사위 지급/교환 (픽카드, 드로우, 히든)
12. **특수 주사위 이벤트 지급**: 쿠키가 다가와서 지급하는 연출

#### 주사위 도전 (메인 화면)
13. **주사위 도전 아이콘**: 1주자 선택 후 메인 화면에 생성
14. **탭 구성**: 하단 [주사위 합성], [주사위 미션] 탭
15. **디폴트 탭 설정**: 미션 진행 중이면 [주사위 미션] 탭이 기본
16. **특수 주사위 버튼**: 좌측 버튼으로 특수 주사위 정보 안내
17. **주사위 인벤토리**: 탭 변경과 상관없이 고정 노출
18. **인벤토리 구분**: 일반 주사위와 특수 주사위 구분 배치

#### 주사위 합성
19. **합성 메뉴 구성**: 다시 굴리기 / 눈 +1 / 특수 주사위 받기 3개 메뉴
20. **다시 굴리기 (리롤)**: 굴려진 주사위를 드래그하여 슬롯 등록 → 리롤
21. **리롤 대상**: 일반 주사위, 특수 주사위 모두 가능
22. **리롤 연출**: 드래그 → 슬롯 이동 → 반짝이는 연출 → 보상 획득 연출
23. **리롤 결과**: 동일한 수가 재차 나올 수 있음
24. **리롤 제한 횟수**: 주자별 최대 횟수 존재
25. **리롤 횟수 초기화**: 다음 주자로 변경 시 초기화 (이월 불가)
26. **리롤 횟수 소진 시**: 슬롯 비활성화
27. **리롤 횟수 추가**: 주사위 미션 6단계 모두 클리어 시 +N회 추가 버프
28. **(합성) 눈 +1**: 동일 눈 주사위 2개 → 눈 +1 주사위 1개 획득
29. **눈 +1 사용 제한**: 일반 주사위만 가능 (특수 주사위 불가)
30. **6+6 합성 불가**: 일반 주사위 6+6은 진행 불가
31. **눈 +1 슬롯 안내**: 1개 장착 시 나머지 슬롯에 같은 눈 반투명 표시
32. **눈 +1 조건 미충족**: "조건에 충족되지 않습니다" 토스트 메시지
33. **(합성) 특수 주사위 받기**: 주사위 2개 합계 10 이상 → 특수 주사위 랜덤 1개
34. **특수 주사위 합성 제한**: 일반 주사위만 가능 (특수 주사위 불가)
35. **특수 주사위 슬롯 안내**: 1개 장착 시 나머지 "N이상" 형태 표시
36. **특수 주사위 조건 미충족**: "조건에 충족되지 않습니다" 토스트 메시지

#### 특수 주사위
37. **특수 주사위 정보 팝업**: 종류, 눈 구성, 합성 시 획득 확률 안내
38. **특수 주사위 구분**: 재질/이미지로 구분 (나무, 메탈, 유리 등)
39. **특수 주사위 표기**: 주사위 눈이 아닌 숫자로 표기
40. **특수 주사위 효과**: 해당 눈(숫자)의 역할만, 추가 효과 없음
41. **특수 주사위 종류**: 나무/블랙/금속(짝수)/유리(홀수)/루비/황금/크리스탈/블랙홀 (8종)
42. **일반+특수 합성**: 일반+특수 눈+1 합성 가능, 결과는 특수 주사위로 지급

#### 주사위 미션 (기본 1~6단계)
43. **미션 단계**: 1~6단계 순차 달성
44. **미션 주사위 등록**: 드래그를 통해 슬롯에 등록
45. **등록 순서**: 순서 상관 없이 원하는 위치에 등록 가능
46. **슬롯 조건 안내**: 조건에 따라 장착 가능한 눈을 반투명 상태로 표시
47. **주사위 소진 시점**: 미션 조건 충족 → 미션 완료 시 소진
48. **미션 완료 전 회수**: 등록 후 완료 전까지 드래그로 인벤토리 회수 가능
49. **회수 안내 텍스트**: "드래그를 통해 등록된 주사위 회수할 수 있습니다"
50. **미션 완료 확인 팝업**: 등록된 주사위가 모두 소진된다는 확인 팝업
51. **미션 완료 연출 플로우**: 확인 팝업 → "누군가가 나타났습니다" 연출 → 보상 확인 팝업 → 보상 획득 팝업
52. **버프 보상 시 플로우**: 보상 확인 팝업 → 버프 획득 팝업
53. **3종 택1 보상 시 플로우**: 보상 확인 팝업 → 버프 선택 화면 팝업 → 버프 획득 팝업
54. **미션 내용 미리 보기**: 1~6단계 미션은 뒤에 배치된 미션 내용 확인 가능
55. **모든 단계 완료 시**: 무작위 미션 단계로 이동

#### 미션 타입별 상세
56. **anyDiceRegister**: 아무 주사위 N개 등록
57. **equalDiceCount**: 같은 눈 주사위 N개 등록
58. **totalDiceValueOver**: N개 주사위 눈 합계 M 초과
59. **straightDiceCount**: 연속된 N개 숫자 등록
60. **fixedDiceCount**: 지정된 눈 주사위 N개 등록 (처음부터 등록 가능 눈 표시)
61. **slotFixedRegister**: 지정된 슬롯에 지정된 주사위 등록 (별도 미션 정보 팝업 필요)

#### 미션 보상
62. **보상 타입 - 모두 지급**: 나열된 보상 모두 지급 (1~3종)
63. **보상 타입 - 선택 보상**: 3종 중 1종 유저 선택
64. **단계별 보상 리스트**: 1단계(레어 버프) ~ 6단계(레전더리 버프 + 리롤 횟수 추가 + 주사위 슬롯 시스템 해금)

#### 무작위 미션
65. **해금 조건**: 주사위 미션 1~6단계 모두 클리어
66. **해금 연출**: "누군가가 다가옵니다" 연출 + 잠금 해제 알림 + [강화 슬롯] 버프 1종 선택
67. **무작위 미션 화면**: 기존 미션 화면과 동일 형태 + [무작위 미션] 버튼 + [미션 변경] 버튼
68. **미션 변경**: 무작위 미션 풀에서 다른 미션으로 변경
69. **미션 변경 예외처리**: 현재와 동일한 미션 미출현
70. **미션 변경 횟수 제한**: 주자별 최대 횟수 제한
71. **미션 변경 연출**: 미션 조건 및 보상 영역 번쩍이는 연출
72. **무작위 미션 팝업**: 미션 이름-조건-보상 아이콘 형태 리스트
73. **보상 아이콘 터치**: 말풍선으로 버프 효과 안내
74. **무작위 미션 보상**: 버프 (난이도에 따라 좋은 버프)

### 중점 확인 사항

#### 1. 주사위 합성 조건 및 제한 사항
- **일반 주사위만 가능** vs **일반+특수 모두 가능** 구분이 합성 타입별로 다름
  - 리롤: 일반+특수 모두 가능
  - 눈 +1: 일반만 가능 (단, 일반+특수 합성 시 결과는 특수 주사위)
  - 특수 주사위 받기: 일반만 가능
- 6+6 합성 불가 예외 처리
- 합성 슬롯 조건 안내 (반투명 표시, "N이상" 표시) 정확성

#### 2. 미션 타입별 슬롯 동작 차이
- `fixedDiceCount`, `slotFixedRegister`는 처음부터 등록 가능한 눈이 지정되어 표시 ([ALL]이 아님)
- `slotFixedRegister`는 텍스트만으로 조건을 알 수 없어 별도 미션 정보 표시 필요
- 각 미션 타입마다 슬롯 조건 안내 방식이 달라 개별 검증 필요

#### 3. 주사위 소진 타이밍 및 회수 가능 여부
- 미션 등록만 하고 완료하지 않은 상태 → 회수 가능
- 미션 완료 시 → 소진 (확인 팝업 후)
- 회수 안내 문구가 "미션 완료 후 회수 가능"으로 오인되지 않도록 주의

#### 4. 리롤 횟수 관리
- 주자별 최대 횟수 존재 → 주자 변경 시 초기화 (이월 불가)
- 미션 6단계 완료 시 추가 횟수 부여
- 횟수 소진 시 슬롯 비활성화

#### 5. 보상 지급 플로우 분기
- 일반 보상: 확인 팝업 → NPC 연출 → 보상 확인 → 보상 획득
- 버프 보상: + 버프 획득 팝업
- 3종 택1 보상: + 버프 선택 화면 → 버프 획득 팝업
- 모두 지급 vs 선택 보상 타입 구분

#### 6. 무작위 미션 해금 및 전환
- 6단계 완료 시 자동 전환 + 주사위 슬롯 잠금 해제 연출
- 미션 변경 시 동일 미션 미출현 예외처리
- 미션 변경 횟수 주자별 제한

#### 7. 플레이 보상의 지연 지급 구조
- 즉시 지급이 아닌 다음 주자 이어달리기 시 지급
- 재도전 시 최고 점수 기준 갱신/유지 로직

#### 8. 눈 +1 합성 시 일반+특수 혼합 케이스
- 기획서에 "일반 주사위만 사용 가능"이라 하면서도 "일반+특수 합성(눈+1) 가능, 결과는 특수"라는 내용이 특수 주사위 섹션에 별도 기재됨
- 이 두 조건 간 정합성 검증 필요 (TC에서 양쪽 모두 커버)
