### 기능 목록

**도움 요청 관련**
1. 도움 요청 가이드 표시 (미클리어 단계에 "도움 요청이 가능하다" 표시, 클리어 완료 단계는 미표시)
2. 도움 요청 버튼 노출 조건 (방장 기준, 해당 단계 클리어 기록 없을 시 노출 / 1회라도 클리어 시 제거)
3. 도움 요청 튜토리얼 (최초 진입 유저 대상 툴팁 + 가이드 팝업 연결)
4. 방 만들기 팝업 (도움 요청 가능한 단계 정보, 도움 요청 빠른 방 만들기 버튼, 일반 방 만들기 버튼)
5. 도움 요청 빠른 방 만들기 터치 시 도움 요청 가이드 팝업 표시
6. 방 내부 도움 요청 버튼 (일반 방에서 도움 요청 활성화/비활성화 토글)
7. 도움 요청 활성화 시 확인 팝업 출력
8. 도움 요청 상태 활성화 후 방이 "도움 요청 진행중인 방"으로 변경
9. 도움 요청 방 생성 후 방장 이탈 시 일반 방으로 전환
10. 방 설정 변경 (도움 요청 전환 후 방 설정 변경 불가, 배틀 버튼 비활성화 등)

**도움 주기 관련 (도움 제공자 측)**
11. 채팅 알림 (실시간 도움 요청 진입 시 "누군가 도움을 요청합니다" 채팅 헤더 표시)
12. 도움주기 탭 버튼 (어비스 채팅 내 도움 요청 목록 탭)
13. 도움 요청 목록 (도움 요청 시간, 현재 참여 인원, 난이도, 단계 정보 표시)
14. 방장 보기 버튼 → 유저 정보 팝업 (닉네임, 전투력 등)
15. 바로가기 버튼 → 방 즉시 진입
16. 바로가기 진입 불가 조건 (이미 다른 방 참여 중 / 방 인원 초과 / 해당 단계 진입 불가 시 토스트 메시지)
17. 방 목록에서 도움 요청중인 방 강조 테두리 + "도움 요청중" 태그 표시
18. 도움 주기 가능 조건 (해당 단계 최초 클리어 기록 보유)
19. 도움 주기 성공 조건 (4페이즈 클리어 시 성공 카운트)
20. 도움 주기 실패 조건 (방장이 튕겨서 클리어 실패 시 실패 처리)

**도움 주기 보상 관련**
21. 도움 주기 성공 시 추가 보상 획득 (결과 화면에서 즉시 획득)
22. 결과 화면에서 [도움] 태그로 보상 정보 별도 표시
23. 도움 주기 보상 주간 최대치 제한 (주간 limit)
24. 보상 리셋 시간 (어비스 주간 보상 리셋 타이밍과 동일)
25. 도움 주기 보상 설정 UI (상단 영역 보상 관련 설정 표시)

**누적/업적/칭호**
26. 누적 도움주기 횟수 표시 (1회 이상 진행 시)
27. 도움 주기 칭호 획득 (누적 횟수 기반)
28. 업적 내 도움 주기 관련 업적 항목

**가이드북**
29. 어비스 레이드 튜토리얼 종료 후 도움 주기/도움 요청 가이드북 펼침
30. 이미 튜토리얼 종료된 유저는 로비 진입 즉시 가이드북 보여짐 (최초 1회)
31. 도움 요청 가이드 페이지 (채팅, 방 목록 안내)
32. 도움 주기 가이드 페이지 (채팅, 방 진입, 보상 안내)

### 중점 확인 사항

1. **도움 요청/주기 핵심 플로우 완결성**: 도움 요청 → 채팅 알림 → 목록 노출 → 방 진입 → 4페이즈 클리어 → 보상 지급까지의 전체 플로우가 끊김 없이 동작하는지 집중 검증

2. **보상 지급 정확성**: 도움 주기 성공 시 추가 보상이 올바르게 지급되는지, [도움] 태그로 별도 표시되는지, 주간 limit 초과 시 보상 미지급이 정확히 동작하는지 (재화 관련 Critical)

3. **방 상태 전환 정합성**: 일반 방 ↔ 도움 요청 방 전환 시 방 설정 변경 제한, 배틀 버튼 상태, 방장 이탈 시 일반 방 전환 등 상태 변경이 정확한지

4. **진입 불가 조건 처리**: 이미 다른 방 참여 중 / 방 인원 초과 / 해당 단계 미진입 가능 / 도움 요청 방이 이미 사라진 경우 등 각 예외 상황에서 적절한 토스트 메시지가 출력되는지

5. **도움 주기 가능 조건 검증**: 해당 단계 최초 클리어 기록이 있는 유저만 도움 주기가 가능한지, 도움을 받아 최초 클리어한 유저도 도움 주기 가능 상태가 되는지

6. **방장 이탈 예외 처리**: 도움 요청 방에서 방장(도움 요청 유저)이 튕길 경우 도움 주기 실패 처리가 정확히 되는지

7. **가이드북 노출 조건**: 튜토리얼 종료 후 가이드북 자동 펼침, 기존 유저 로비 진입 시 최초 1회 노출이 정확히 동작하는지

8. **주간 리셋 타이밍**: 도움 주기 보상 리셋 시간이 어비스 주간 보상 리셋과 동일하게 적용되는지
