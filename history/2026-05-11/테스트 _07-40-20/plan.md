### 중점 확인 사항

1. **제거 기능 검증의 명확화**
   - 기획서에서 명시한 3가지 제거 기능(코코방울의 축복 / 트로피 상자 / 무지개 상자 지급 이벤트)이 챌린지 레이드 모드에서 노출되지 않는지를 **반드시 별도의 TC**로 작성한다.

2. **다시하기 버튼의 재화 소모**
   - 페니콩 1 소모가 표기되어 있으므로 재화 차감 동작은 Critical로 분류하여 작성한다.

3. **트로피 획득/미획득 상태의 분리**
   - 동일 UI 영역이지만 "획득 상태(활성)"와 "미획득 상태(회색)"는 결과가 다르므로 각각 분리하여 작성한다.

4. **추측 금지 항목 제외**
   - 회색 트로피 2종(P 트로피, 일반 트로피)은 정확한 명칭/조건이 명시되지 않았으므로 "미획득 트로피"로 통칭하여 작성한다.
   - "강적 발견" 배너 텍스트, 획득 아이템의 정확한 종류 등 시각적으로만 식별 가능한 세부 명칭은 일반화하여 표현한다.

5. **화면 이동(나가기) TC 배치**
   - 화면 이동 케이스는 클리어 화면 카테고리 마지막에 배치한다.

6. **기존 기능 동일성 검증**
   - "이 외의 기능은 기존 기능과 동일"이라는 명시가 있으나, 기존 기능의 구체 명세가 없어 추측 작성은 지양하고 이미지로 확인 가능한 UI 요소만 작성한다.
